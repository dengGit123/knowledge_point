# CSS 边框背景图

> 官方文档：[MDN - border-image](https://developer.mozilla.org/zh-CN/docs/Web/CSS/border-image)

## 什么是边框背景图？

**边框背景图 = 用图片代替纯色边框，创建更丰富的边框效果**

普通边框只能用纯色，而 border-image 可以：
- 🖼️ 用图片做边框
- 🌈 用渐变做边框
- 🎨 创建不规则边框效果
- 📐 实现花边、相框等装饰效果

```
普通边框：                边框背景图：
┌─────────────┐          ╔═══════════╗
│             │          ║  ╱‾╲  ╱‾╲  ║
│   纯色边框   │          ║ ╱    ╲╱    ╲ ║
│             │          ║            ║
└─────────────┘          ╚═══════════╝
```

## 一、border-image 基本语法

### 1.1 完整简写

```css
border-image: source slice / width / outset repeat;
```

```css
.box {
  border: 20px solid;
  border-image: url('border.png') 30 / 20px / 0 round;
  /*             │          │   │    │   └─ 重复方式 */
  /*             │          │   │    └───── 外边距 */
  /*             │          │   └────────── 边框宽度 */
  /*             │          └────────────── 切割位置 */
  /*             └───────────────────────── 图片源 */
}
```

### 1.2 最简单用法

```css
.box {
  border: 20px solid;
  border-image: url('border.png') 30 round;
}
```

## 二、border-image-source - 图片源

```css
/* 使用图片 */
border-image-source: url('border.png');
border-image-source: url('./images/frame.png');
border-image-source: url('https://example.com/border.jpg');

/* 使用渐变 */
border-image-source: linear-gradient(to right, red, blue);
border-image-source: radial-gradient(circle, red, blue);
border-image-source: conic-gradient(red, blue, green);

/* 多个图片（很少用）*/
border-image-source: url('border1.png'), url('border2.png');
```

## 三、border-image-slice - 图片切割（核心）

### 3.1 基本概念

将图片切割成 9 个区域，4 个角、4 条边、1 个中间：

```
原始图片（假设 90x90）:
┌─────────────────────────────┐
│        上 (30px)            │
│    ┌───────────┐            │
│左  │  1  │  2  │  右        │
│(30)├─────┼─────┤ (30)       │
│    │  3  │  4  │            │
│    └───────────┘            │
│        下 (30px)            │
└─────────────────────────────┘

切割后应用到边框：
┌─────────────────────────────┐
│ 1 │       2        │ 3      │  ← 上边
├───┼────────────────┼────────┤
│   │                │        │  ← 中间
│ 4 │      内容      │   5    │
│   │                │        │
├───┼────────────────┼────────┤
│ 6 │       7        │ 8      │  ← 下边
└─────────────────────────────┘

1,3,6,8 = 四角（保持原样，不变形）
2,4,5,7 = 边缘（按规则重复）
9 = 中间（可选显示）
```

### 3.2 切割语法

```css
/* 单个值：四个方向相同 */
border-image-slice: 30;           /* 30px */
border-image-slice: 25%;          /* 25% */

/* 两个值：上下 左右 */
border-image-slice: 30 20;        /* 上下30 左右20 */

/* 三个值：上 左右 下 */
border-image-slice: 30 20 40;     /* 上30 左右20 下40 */

/* 四个值：上 右 下 左 */
border-image-slice: 30 20 40 25;  /* 上30 右20 下40 左25 */

/* fill 关键字：显示中间区域 */
border-image-slice: 30 fill;      /* 中间区域也会显示 */
```

### 3.3 像素 vs 百分比

```css
/* 数字 = 像素值 */
border-image-slice: 30;           /* 30px */

/* 百分比 = 相对于图片尺寸 */
border-image-slice: 33.33%;       /* 图片的三分之一 */

/* 混合使用（很少见）*/
border-image-slice: 30 20% 40 15%;
```

### 3.4 实际例子

假设有一张 120x120 的边框图片：

```css
/* 均匀切割 */
.box {
  border-image-slice: 40;         /* 每边 40px */
}

/* 效果：四个角各 40x40，不变形拉伸 */

/* 只切边缘 */
.box {
  border-image-slice: 10;         /* 只切 10px 边缘 */
}

/* 效果：四个角很小，边框主要靠拉伸 */
```

## 四、border-image-width - 边框宽度

```css
/* 具体值 */
border-image-width: 20px;
border-image-width: 1em;
border-image-width: 0.5cm;

/* 相对值 */
border-image-width: 10%;          /* 相对于边框图像区域 */
border-image-width: 1;            /* 相对于边框宽度（倍数）*/

/* 四边不同 */
border-image-width: 20px 10px 20px 10px;

/* auto（使用 slice 值）*/
border-image-width: auto;
```

**与 border-width 的关系：**

```css
.box {
  border: 20px solid;             /* 实际占据的空间 */
  border-image-width: 30px;       /* 图片显示的宽度（可以超过 border）*/
}

/* 简化：只设置 border-width */
.box {
  border: 20px solid;
  border-image: url('border.png') 30 / 20px round;
  /*                                              ↑ 不用设，自动用 border 宽度 */
}
```

## 五、border-image-outset - 外边距

让边框图片向外扩展（不占用布局空间）：

```css
/* 单个值 */
border-image-outset: 10px;        /* 向外扩展 10px */

/* 四个值 */
border-image-outset: 10px 20px 10px 20px;

/* 相对值 */
border-image-outset: 1;           /* border-width 的倍数 */

/* 效果：边框图片向外扩展，但不影响元素尺寸 */
```

**示意图：**
```
border-image-outset: 10px

     ← 10px 扩展 →
┌────────────────────┐
│ ╔════════════════╗ │
│ ║  元素内容       ║ │
│ ╚════════════════╝ │
└────────────────────┘
  ↑ 元素实际边界
```

## 六、border-image-repeat - 重复方式

### 6.1 四种方式

```css
/* stretch（默认）：拉伸 */
border-image-repeat: stretch;

/* repeat：重复（可能裁剪）*/
border-image-repeat: repeat;

/* round：重复（自动调整，不裁剪）*/
border-image-repeat: round;

/* space：重复（均匀分布）*/
border-image-repeat: space;

/* 水平和垂直分别设置 */
border-image-repeat: round stretch;
```

### 6.2 效果对比

```
stretch（拉伸）:
┌─────────────────┐
│ ╱───────stretch │
│ ╲              │
│  ╲_____________ │
└─────────────────┘

repeat（重复，可能裁剪）:
┌─────────────────┐
│ ABABCABABCABCAB │
│ C              C │
│ C_____________ C │
└─────────────────┘

round（重复，自动调整）:
┌─────────────────┐
│ ABABABABA...AB │
│ A              A │
│ B_____________ B │
└─────────────────┘

space（重复，均匀分布）:
┌─────────────────┐
│ A  B  A  B  A  B │
│ A              A │
│ B_____________ B │
└─────────────────┘
```

## 七、渐变边框

### 7.1 线性渐变边框

```css
.box {
  border: 10px solid;
  border-image: linear-gradient(to right, #3498db, #e74c3c) 1;
}

/* 多色渐变 */
.box {
  border: 15px solid;
  border-image: linear-gradient(
    45deg,
    #3498db,
    #e74c3c,
    #f39c12,
    #2ecc71
  ) 1;
}
```

### 7.2 径向渐变边框

```css
.box {
  border: 20px solid;
  border-image: radial-gradient(circle, #3498db, transparent) 1;
}
```

### 7.3 锥形渐变边框

```css
.box {
  border: 15px solid;
  border-image: conic-gradient(
    from 0deg,
    red,
    yellow,
    green,
    blue,
    red
  ) 1;
}
```

### 7.4 圆角渐变边框（特殊处理）

**⚠️ border-image 不支持 border-radius！**

```css
/* ❌ 这样不行 */
.box {
  border: 10px solid;
  border-image: linear-gradient(to right, red, blue) 1;
  border-radius: 20px;  /* 无效！*/
}

/* ✅ 使用伪元素实现圆角渐变边框 */
.box {
  position: relative;
  border-radius: 20px;
  background: white;
  padding: 20px;
}

.box::before {
  content: '';
  position: absolute;
  inset: -10px;  /* 边框宽度 */
  border-radius: inherit;
  padding: 10px;
  background: linear-gradient(to right, red, blue);
  -webkit-mask:
    linear-gradient(#fff 0 0) content-box,
    linear-gradient(#fff 0 0);
  -webkit-mask-composite: xor;
  mask-composite: exclude;
  pointer-events: none;
}

/* ✅ 或用 background-clip 方式 */
.box {
  border: 10px solid transparent;
  border-radius: 20px;
  background:
    linear-gradient(white, white) padding-box,
    linear-gradient(to right, red, blue) border-box;
}
```

## 十、常见问题

### Q1：border-image 和 border-radius 冲突？

```css
/* ❌ 不支持圆角 */
.box {
  border-image: url('border.png') 30;
  border-radius: 20px;  /* 无效 */
}

/* ✅ 解决：使用 background-clip */
.box {
  border: 10px solid transparent;
  border-radius: 20px;
  background:
    url('border.png') border-box,
    white padding-box;
  background-origin: border-box;
}

/* ✅ 或用伪元素 */
.box {
  position: relative;
  border-radius: 20px;
}
.box::before {
  /* 伪元素实现圆角边框 */
}
```

### Q2：边框图片不显示？

```css
/* 检查清单 */
.box {
  /* 1. 是否设置了 border-width？*/
  border: 20px solid;  /* ✅ 必须设置 */

  /* 2. border-style 不能是 none */
  border-style: solid;  /* ✅ 可以是 solid，虽然会被图片覆盖 */

  /* 3. 图片路径是否正确 */
  border-image-source: url('correct-path.png');
}
```

### Q3：边框图片变形了？

```css
/* 调整 slice 值 */
.box {
  border-image-slice: 30;      /* 尝试不同的值 */
}

/* 或者用 round 代替 stretch */
.box {
  border-image-repeat: round;  /* 保持比例重复 */
}
```

### 简写速查

```css
/* 最简写法 */
border-image: url('img.png') 30 round;

/* 完整写法 */
border-image: url('img.png') 30 / 20px / 0 round;
/*              │        │   │     │   └─ repeat */
/*              │        │   │     └────── outset */
/*              │        │   └──────────── width */
/*              │        └──────────────── slice */
/*              └───────────────────────── source */

/* 渐变边框 */
border-image: linear-gradient(to right, red, blue) 1;
```

### repeat 值选择

| 值 | 效果 | 适用场景 |
|-----|------|---------|
| `stretch` | 拉伸 | 简单边框 |
| `repeat` | 重复（可能裁剪）| 图案边框 |
| `round` | 重复（自动调整）| 图案边框（推荐）|
| `space` | 重复（均匀分布）| 需要精确间距 |
