# 表格布局

## 表格相关 display 值

| 值 | 说明 | 对应 HTML 标签 |
|----|------|----------------|
| `table` | 表格容器 | `<table>` |
| `inline-table` | 内联表格 | - |
| `table-caption` | 表格标题 | `<caption>` |
| `table-row` | 表格行 | `<tr>` |
| `table-row-group` | 行组 | `<tbody>` |
| `table-header-group` | 头部行组 | `<thead>` |
| `table-footer-group` | 底部行组 | `<tfoot>` |
| `table-column` | 列 | `<col>` |
| `table-column-group` | 列组 | `<colgroup>` |
| `table-cell` | 单元格 | `<th>`, `<td>` |

## table-layout

控制表格布局算法。

### 语法

```css
table-layout: auto | fixed;
```

### auto（默认）

- **含义**：列宽由内容决定
- **特点**：
  - 需要读取完所有表格内容才能确定布局
  - 单元格内容不受宽度限制
  - 较慢，但更灵活

```html
<table style="table-layout: auto; width: 100%;">
  <tr>
    <td>很长的内容会撑开单元格</td>
    <td>短内容</td>
  </tr>
</table>
```

### fixed

- **含义**：列宽由表格宽度或第一行决定
- **特点**：
  - 只需读取第一行即可确定布局
  - 内容超出会被截断或换行
  - 更快，性能更好
  - 可通过 `col` 或第一行单元格的 `width` 控制列宽

```html
<table style="table-layout: fixed; width: 600px;">
  <colgroup>
    <col style="width: 100px;">
    <col style="width: 200px;">
    <col>
  </colgroup>
  <tr>
    <td>固定 100px</td>
    <td>固定 200px</td>
    <td>剩余空间</td>
  </tr>
</table>
```

### 场景

| 场景 | 推荐值 |
|------|--------|
| 大数据表格（1000+ 行）| `fixed` |
| 列宽固定 | `fixed` |
| 内容长度差异大 | `auto` |
| 需要响应式宽度 | `auto` 或 `fixed` + 百分比 |

## 列宽控制

### 方式一：col / colgroup（推荐）

```html
<table style="table-layout: fixed; width: 100%;">
  <colgroup>
    <col style="width: 100px;">      <!-- 固定宽度 -->
    <col style="width: 20%;">        <!-- 百分比 -->
    <col style="width: 3fr;">        <!-- 比例（Chrome 111+） -->
    <col>                            <!-- 自动分配剩余 -->
  </colgroup>
  <tr>
    <td>100px</td>
    <td>20%</td>
    <td>3份</td>
    <td>剩余</td>
  </tr>
</table>
```

### 方式二：第一行单元格宽度

```html
<table style="table-layout: fixed; width: 600px;">
  <tr>
    <th style="width: 100px;">标题1</th>
    <th style="width: 200px;">标题2</th>
    <th>自动</th>
  </tr>
  <tr>
    <td>内容</td>
    <td>内容</td>
    <td>内容</td>
  </tr>
</table>
```

### 方式三：HTML 属性（不推荐）

```html
<table>
  <col width="100">
  <col width="20%">
  <tr>
    <td>...</td>
  </tr>
</table>
```

### 列宽计算规则

#### table-layout: auto

1. 读取所有单元格内容
2. 根据内容长度、设置宽度、百分比自动分配
3. 可能超出设置的表格宽度

```css
table {
  table-layout: auto;
  width: 500px;  /* 可能被内容撑开 */
}
```

#### table-layout: fixed（推荐）

1. 只看第一行（或 colgroup）的宽度声明
2. 按设置宽度分配，剩余空间平均分配给未设置的列
3. 内容超出会换行或隐藏

```css
table {
  table-layout: fixed;
  width: 100%;
}

/* 列宽分配示例：
   第1列: 100px
   第2列: 200px
   第3列: 剩余空间全部
*/
```

### 宽度单位

| 单位 | 说明 | 示例 |
|------|------|------|
| `px` | 固定像素 | `width: 100px` |
| `%` | 表格宽度的百分比 | `width: 20%` |
| `fr` | 分数单位（Chrome 111+）| `width: 2fr` |
| `auto` | 自动（默认） | `width: auto` |

### min-width / max-width

```css
td {
  min-width: 50px;   /* 最小宽度 */
  max-width: 300px;  /* 最大宽度 */
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

### 实用场景

#### 固定首列，其余自适应

```css
table {
  table-layout: fixed;
  width: 100%;
}
col:first-child {
  width: 80px;
}
```

#### 操作列固定宽度

```html
<table>
  <colgroup>
    <col>                     <!-- 自适应 -->
    <col>                     <!-- 自适应 -->
    <col style="width: 100px;"> <!-- 操作列 -->
  </colgroup>
</table>
```

#### 均分列宽

```css
/* 所有列等宽 */
table {
  table-layout: fixed;
  width: 100%;
}
td, th {
  width: 1%;  /* 触发均分 */
}
```

#### 按比例分配

```html
<!-- 1:2:3 比例 -->
<table style="table-layout: fixed; width: 100%;">
  <colgroup>
    <col style="width: 1fr;">  <!-- 支持 Chrome 111+ -->
    <col style="width: 2fr;">
    <col style="width: 3fr;">
  </colgroup>
</table>

<!-- 兼容写法：用百分比 -->
<colgroup>
  <col style="width: 16.66%;">   <!-- 1/6 -->
  <col style="width: 33.33%;">   <!-- 2/6 -->
  <col style="width: 50%;">      <!-- 3/6 -->
</colgroup>
```

## border-collapse

控制表格边框合并方式。

### 语法

```css
border-collapse: collapse | separate;
```

### collapse

- **含义**：相邻边框合并为一条
- **效果**：单线边框，单元格间无间隙
- **注意**：`border-spacing` 和 `empty-cells` 无效

```css
table {
  border-collapse: collapse;
}
td, th {
  border: 1px solid #ddd;
}
```

### separate（默认）

- **含义**：每个单元格独立边框
- **效果**：双线边框，单元格间有间隙

```css
table {
  border-collapse: separate;
  border-spacing: 2px; /* 间隙 */
}
```

## border-spacing

设置单元格之间的间距。

### 语法

```css
border-spacing: <length> | <length-horizontal> <length-vertical>;
```

### 示例

```css
/* 水平和垂直都是 5px */
border-spacing: 5px;

/* 水平 10px，垂直 5px */
border-spacing: 10px 5px;

/* 继承 */
border-spacing: inherit;
```

### 注意

- 只在 `border-collapse: separate` 时有效
- 不能为负值
- 不影响表格外边距

## caption-side

设置表格标题的位置。

### 语法

```css
caption-side: top | bottom | block-start | block-end;
```

### 示例

```css
caption {
  caption-side: bottom;
  text-align: center;
  font-weight: bold;
}
```

```
┌─────────────────┐
│  表格内容         │
├─────────────────┤
│  数据行          │
└─────────────────┘
    表格标题
```

## empty-cells

控制空单元格是否显示边框。

### 语法

```css
empty-cells: show | hide;
```

### show（默认）

空单元格显示边框和背景。

### hide

空单元格不显示边框和背景。

### 注意

- 只在 `border-collapse: separate` 时有效
- 单元格需完全无内容（包括空白符）

```html
<table style="border-collapse: separate; empty-cells: hide;">
  <tr>
    <td>有内容</td>
    <td></td>  <!-- 无边框 -->
  </tr>
</table>
```

## vertical-align（表格）

表格单元格中垂直对齐的特殊值。

### 语法

```css
vertical-align: baseline | top | middle | bottom | sub | super | text-top | text-bottom;
```

### 表格常用值

| 值 | 说明 |
|----|------|
| `baseline` | 与第一行文本基线对齐 |
| `top` | 与行顶对齐 |
| `middle` | 行内垂直居中 |
| `bottom` | 与行底对齐 |

```css
td {
  height: 50px;
  vertical-align: middle;
}
```

## 表格布局实现居中

### 水平垂直居中

```css
/* 父元素 */
.container {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
  width: 300px;
  height: 300px;
}
```

### 经典居中方案

```html
<div style="display: table; width: 100%; height: 100vh;">
  <div style="display: table-cell; vertical-align: middle; text-align: center;">
    <div>居中内容</div>
  </div>
</div>
```

## 响应式表格

### 水平滚动

```css
.table-wrapper {
  overflow-x: auto;
  -webkit-overflow-scrolling: touch;
}

table {
  min-width: 600px; /* 确保在小屏幕上可滚动 */
}
```

### 卡片式堆叠（移动端）

```css
@media (max-width: 600px) {
  table, thead, tbody, th, td, tr {
    display: block;
  }

  thead tr {
    position: absolute;
    top: -9999px;
    left: -9999px;
  }

  tr {
    margin-bottom: 1rem;
    border: 1px solid #ccc;
  }

  td {
    border: none;
    position: relative;
    padding-left: 50%;
  }

  td:before {
    position: absolute;
    left: 10px;
    content: attr(data-label);

```

## 性能优化

### 大数据表格

```css
table {
  table-layout: fixed; /* 关键：提升渲染性能 */
  width: 100%;
}

/* 使用 will-change 优化滚动 */
tbody {
  will-change: transform;
}
```

表格数据量大时，使用虚拟滚动（只渲染可见行）：
- react-window
- vue-virtual-scroller

## 表格样式技巧

### 斑马纹（隔行变色）

```css
tbody tr:nth-child(odd) {
  background: #f9f9f9;
}

/* 或者偶数行 */
tbody tr:nth-child(even) {
  background: #f5f5f5;
}
```

### 固定表头

```css
table {
  table-layout: fixed;
  width: 100%;
}

thead {
  position: sticky;
  top: 0;
  z-index: 10;
  background: #fff;
}

/* 固定首列 */
tbody td:first-child {
  position: sticky;
  left: 0;
  background: #fff;
  z-index: 5;
}

/* 固定表头 + 首列（左上角单元格）*/
thead th:first-child {
  position: sticky;
  top: 0;
  left: 0;
  z-index: 15;
}
```

### 文本溢出省略

```css
td {
  max-width: 200px;
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}

/* 多行省略 */
td {
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
  overflow: hidden;
}
```

### 圆角表格

```css
table {
  border-collapse: separate;
  border-spacing: 0;
  border-radius: 8px;
  overflow: hidden;
}

/* 单独控制四个角 */
table tr:first-child td:first-child {
  border-top-left-radius: 8px;
}
table tr:first-child td:last-child {
  border-top-right-radius: 8px;
}
table tr:last-child td:first-child {
  border-bottom-left-radius: 8px;
}
table tr:last-child td:last-child {
  border-bottom-right-radius: 8px;
}
```

### 紧凑型表格

```css
table {
  table-layout: fixed;
  width: 100%;
  font-size: 14px;
}

td, th {
  padding: 8px 12px;
  line-height: 1.5;
}

/* 移除边框，仅用分隔线 */
td, th {
  border: none;
  border-bottom: 1px solid #eee;
}
```

## 单元格合并

### colspan（跨列）

```html
<table>
  <tr>
    <td colspan="2">跨两列</td>
    <td>普通单元格</td>
  </tr>
  <tr>
    <td>1</td>
    <td>2</td>
    <td>3</td>
  </tr>
</table>
```

### rowspan（跨行）

```html
<table>
  <tr>
    <td rowspan="2">跨两行</td>
    <td>1</td>
  </tr>
  <tr>
    <td>2</td>
  </tr>
</table>
```

### 合并时的列宽计算

```html
<table style="table-layout: fixed; width: 600px;">
  <colgroup>
    <col style="width: 100px;">
    <col style="width: 200px;">
    <col style="width: 300px;">
  </colgroup>
  <tr>
    <!-- colspan 的宽度 = 合并列的总和 -->
    <td colspan="2">宽度为 100px + 200px = 300px</td>
    <td>300px</td>
  </tr>
</table>
```

## 浏览器兼容性

| 特性 | 兼容性 | 备注 |
|------|--------|------|
| `table-layout: fixed` | 全支持 | IE+ |
| `border-collapse` | 全支持 | IE+ |
| `fr` 单位 | Chrome 111+ | 避免在生产使用 |
| `position: sticky` 表头 | 现代浏览器 | iOS Safari 需要关注 |
| `col` 样式 | 部分支持 | 只能设置 width, border, background |

**注意**：`<col>` 元素样式限制

```css
/* ❌ 不生效 */
col {
  padding: 10px;      /* 无效 */
  height: 50px;       /* 无效 */
  content: "...";     /* 无效 */
}

/* ✅ 生效 */
col {
  width: 100px;       /* 有效 */
  border: 1px solid;  /* 有效 */
  background: #f00;   /* 有效 */
  visibility: collapse;/* 有效 */
}
```

## 表格布局 vs 其他布局

| 场景 | 推荐方案 | 原因 |
|------|----------|------|
| 二维数据展示 | `<table>` | 语义正确，可访问性好 |
| 表单布局 | Flex / Grid | 更灵活 |
| 整体页面布局 | Grid / Flex | 现代布局方式 |
| 简单的行列对齐 | `display: table-cell` | 垂直居中方便 |
| 卡片网格 | Grid | 响应式更方便 |

### 使用 display: table 模拟表格

```css
.table-like {
  display: table;
  table-layout: fixed;
  width: 100%;
}

.row {
  display: table-row;
}

.cell {
  display: table-cell;
  vertical-align: middle;
}
```

## 常见问题

### Q: 单元格宽度不生效？

确保：
1. 使用 `table-layout: fixed`
2. 在 `<col>` 或第一行单元格设置宽度
3. 表格本身有明确宽度

### Q: 表格边框有间隙？

```css
table {
  border-collapse: collapse;
}
```

### Q: 表格无法使用 margin？

```css
/* 使用包裹元素 */
.table-wrapper {
  margin: 20px;
}
```

### Q: hover 行效果不连贯？

```css
tbody tr:hover {
  background: #f5f5f5;
}

/* 边框合并模式下 */
table {
  border-collapse: collapse;
}
```

### Q: 内容过长撑开表格？

```css
table {
  table-layout: fixed;  /* 关键 */
  word-break: break-all;
}

td {
  overflow: hidden;
  text-overflow: ellipsis;
}
```

### Q: 固定表头在 iOS 不工作？

```css
thead {
  position: -webkit-sticky; /* iOS Safari */
  position: sticky;
  top: 0;
  z-index: 10;
}

/* 确保父容器有明确高度 */
.table-wrapper {
  height: 500px;
  overflow-y: auto;
  -webkit-overflow-scrolling: touch;
}
```

### Q: 合并单元格后样式错乱？

确保合并后单元格数量一致：

```html
<!-- ❌ 错误：第二行只有2个单元格 -->
<tr>
  <td colspan="2">A</td>
  <td>B</td>
</tr>
<tr>
  <td>C</td>  <!-- 缺少一个单元格 -->
  <td>D</td>
</tr>

<!-- ✅ 正确 -->
<tr>
  <td colspan="2">A</td>
  <td>B</td>
</tr>
<tr>
  <td colspan="2">C</td>
  <td>D</td>
</tr>
```

### Q: col 的 background 不显示？

```css
/* col 的 background 可能被 td/th 覆盖 */
table {
  border-collapse: separate;  /* separate 下更可靠 */
}

/* 或直接给 td 设置 */
td:nth-child(1) {
  background: #f5f5f5;
}
```
