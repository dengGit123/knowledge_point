# CSS 背景图

> 官方文档：[MDN - background](https://developer.mozilla.org/zh-CN/docs/Web/CSS/background)

## 什么是背景图？

**背景图 = 给元素"贴"一张图片，作为装饰背景**

想象一下：
- 背景图就像房间的**墙纸**，铺在里面
- 内容（文字、图片）就像**家具**，放在墙纸前面
- 背景图不影响内容布局，只是装饰

```
┌─────────────────────────────────┐
│  ┌─────────────────────────┐    │
│  │   背景图（墙纸）         │    │
│  │  ┌─────────────────┐    │    │
│  │  │   内容（家具）    │    │    │
│  │  │   文字、图片     │    │    │
│  │  └─────────────────┘    │    │
│  └─────────────────────────┘    │
└─────────────────────────────────┘
```

## 一、基本属性

### 1.1 background-image - 设置背景图

```css
.box {
  background-image: url('image.jpg');
  background-image: url('./images/logo.png');
  background-image: url('https://example.com/image.jpg');
}

/* 多个背景图（逗号分隔，前者在上）*/
.box {
  background-image:
    url('overlay.png'),
    url('background.jpg');
}
```

### 1.2 background-color - 背景色

```css
.box {
  background-color: red;
  background-color: #ff0000;
  background-color: rgb(255, 0, 0);
  background-color: rgba(255, 0, 0, 0.5);  /* 半透明 */
}

/* 背景色会显示在背景图下面 */
.box {
  background-color: #333;
  background-image: url('pattern.png');
}
```

> 💡 **背景图加载失败时，背景色会显示**

### 1.3 background-repeat - 重复方式

```css
/* 默认：水平和垂直都重复 */
background-repeat: repeat;

/* 水平重复 */
background-repeat: repeat-x;

/* 垂直重复 */
background-repeat: repeat-y;

/* 不重复 */
background-repeat: no-repeat;

/* 简写：水平 垂直 */
background-repeat: repeat no-repeat;   /* 水平重复，垂直不重复 */
background-repeat: space;              /* 均匀分布，不裁剪 */
background-repeat: round;              /* 缩放后完整显示 */
```

**效果对比：**
```
repeat:        repeat-x:      no-repeat:
┌─────┐        ┌─────────┐    ┌─────┐
│图片图│        │图片图片图片│    │图片 │
│片图片│        │         │    │     │
│图片图│        └─────────┘    │     │
└─────┘                      └─────┘
```

### 1.4 background-position - 位置

```css
/* 关键字 */
background-position: center;
background-position: top left;
background-position: bottom right;

/* 百分比 */
background-position: 50% 50%;      /* 水平50% 垂直50% = center */
background-position: 0% 0%;         /* 左上角 */
background-position: 100% 100%;     /* 右下角 */

/* 具体值 */
background-position: 10px 20px;     /* 距左10px 距上20px */
background-position: 10px center;   /* 混合写法 */

/* 多背景图位置 */
background-position:
  0 0,
  50px 50px,
  center;
```

**百分比计算规则：**
```
0% 0%    = 左上角
50% 50%  = 居中（图片中心与容器中心对齐）
100% 100% = 右下角
```

### 1.5 background-size - 尺寸

```css
/* 具体值 */
background-size: 300px 200px;       /* 宽300px 高200px */
background-size: 100% 50%;          /* 宽100% 高50% */

/* 关键字 */
background-size: cover;             /* 覆盖整个容器（可能裁剪）*/
background-size: contain;           /* 完整显示图片（可能有留白）*/
background-size: auto;              /* 自动（默认值）*/

/* 单值：第二个值默认 auto */
background-size: 100%;              /* 宽100%，高自动 */
background-size: 50%;               /* 缩小到50% */
```

**cover vs contain：**

```
容器（红色框）：

cover（覆盖）:
┌─────────────────┐
│  图片图片图片   │ ← 裁剪了，但填满
│  图片图片图片   │
│  图片图片图片   │
└─────────────────┘

contain（包含）:
┌─────────────────┐
│  ┌─────────┐    │ ← 完整显示
│  │  图片   │    │   但有留白
│  └─────────┘    │
└─────────────────┘
```

### 1.6 background-attachment - 固定方式

```css
/* 默认：随元素滚动 */
background-attachment: scroll;

/* 固定：相对于视口固定 */
background-attachment: fixed;

/* 随元素内容滚动 */
background-attachment: local;
```

**fixed 实现视差滚动：**
```css
.hero {
  background-image: url('bg.jpg');
  background-attachment: fixed;  /* 背景固定不动 */
  background-size: cover;
  height: 100vh;
}
```

### 1.7 background-clip - 裁剪区域

```css
/* 默认：裁剪到边框盒 */
background-clip: border-box;

/* 裁剪到内边距盒 */
background-clip: padding-box;

/* 裁剪到内容区 */
background-clip: content-box;

/* 文字裁剪（文字渐变效果）*/
background-clip: text;
-webkit-background-clip: text;
color: transparent;
```

**文字渐变效果：**
```css
.gradient-text {
  background: linear-gradient(to right, #3498db, #e74c3c);
  background-clip: text;
  -webkit-background-clip: text;
  color: transparent;
}
```

### 1.8 background-origin - 起始位置

```css
/* 从边框开始 */
background-origin: border-box;

/* 从内边距开始（默认）*/
background-origin: padding-box;

/* 从内容区开始 */
background-origin: content-box;
```

**区别对比：**
```
padding-box:          content-box:
┌─────────────────┐   ┌─────────────────┐
│  ╔───────────╗  │   │                 │
│  ║  内容     ║  │   │  ┌─────────┐    │
│  ╚───────────╝  │   │  │  内容   │    │
└─────────────────┘   └─────────────────┘
```

## 二、简写属性

### 2.1 background 完整简写

```css
background:
  [background-color]
  [background-image]
  [background-position] / [background-size]
  [background-repeat]
  [background-attachment]
  [background-clip]
  [background-origin];
```

### 2.2 常用简写示例

```css
/* 最简单 */
background: url('bg.jpg');

/* 常用组合 */
background: url('bg.jpg') no-repeat center;
background: url('bg.jpg') center / cover;
background: #333 url('bg.jpg') no-repeat center;

/* 完整写法 */
background:
  #333 url('bg.jpg')           /* 颜色 + 图片 */
  no-repeat                    /* 不重复 */
  center / cover               /* 居中 + 覆盖 */
  padding-box                  /* 裁剪区域 */
  scroll;                      /* 滚动方式 */

/* 多背景 */
background:
  url('overlay.png') no-repeat center,
  url('bg.jpg') center / cover;
```

### 2.3 简写顺序规则

```css
background:
  url('image.jpg')      /* 1. 图片 */
  center               /* 2. 位置 */
  / cover              /* 3. / 尺寸 */
  no-repeat            /* 4. 重复 */
  fixed                /* 5. 固定方式 */
  padding-box          /* 6. 裁剪 */
  content-box;         /* 7. 起始 */
```

> ⚠️ **注意**：position 和 size 之间用 `/` 分隔

## 三、渐变背景

### 3.1 线性渐变 linear-gradient

```css
/* 基本语法 */
background: linear-gradient(方向, 颜色1, 颜色2);

/* 方向 */
background: linear-gradient(to right, red, blue);      /* 向右 */
background: linear-gradient(to bottom, red, blue);     /* 向下 */
background: linear-gradient(to top right, red, blue);  /* 向右上 */
background: linear-gradient(45deg, red, blue);         /* 45度角 */

/* 多个颜色 */
background: linear-gradient(red, yellow, green, blue);

/* 百分比控制 */
background: linear-gradient(to right, red 0%, blue 100%);
background: linear-gradient(to right, red 20%, blue 80%);
background: linear-gradient(to right, red 0%, red 50%, blue 50%, blue 100%);
/* ↑ 这种写法可以实现硬边过渡 */

/* 重复渐变 */
background: repeating-linear-gradient(
  45deg,
  red,
  red 10px,
  blue 10px,
  blue 20px
);
```

### 3.2 径向渐变 radial-gradient

```css
/* 基本语法 */
background: radial-gradient(形状 大小 位置, 颜色1, 颜色2);

/* 形状 */
background: radial-gradient(circle, red, blue);    /* 圆形 */
background: radial-gradient(ellipse, red, blue);   /* 椭圆 */

/* 大小 */
background: radial-gradient(circle closest-side, red, blue);
background: radial-gradient(circle farthest-corner, red, blue);

/* 位置 */
background: radial-gradient(circle at top left, red, blue);
background: radial-gradient(circle at 50% 50%, red, blue);

/* 多个颜色 */
background: radial-gradient(circle, red, yellow, green, blue);
```

**径向渐变大小关键词：**
```
closest-side:     最近边
closest-corner:   最近角
farthest-side:    最远边
farthest-corner:  最远角（默认）
```

### 3.3 锥形渐变 conic-gradient

```css
/* 基本语法 */
background: conic-gradient(from 角度 at 位置, 颜色1, 颜色2);

/* 简单饼图 */
background: conic-gradient(
  red 0% 25%,
  blue 25% 50%,
  green 50% 75%,
  yellow 75% 100%
);

/* 从顶部开始 */
background: conic-gradient(from 0deg, red, blue);

/* 指定中心 */
background: conic-gradient(at 50% 50%, red, blue);

/* 重复锥形渐变 */
background: repeating-conic-gradient(
  red 0deg 10deg,
  blue 10deg 20deg
);
```

### 3.4 实用渐变效果

```css
/* 虚线效果 */
background: linear-gradient(
  to right,
  #333 50%,
  transparent 50%
);
background-size: 20px 100%;

/* 条纹背景 */
background: repeating-linear-gradient(
  45deg,
  #f0f0f0,
  #f0f0f0 10px,
  #e0e0e0 10px,
  #e0e0e0 20px
);

/* 网格背景 */
background:
  linear-gradient(to right, #ddd 1px, transparent 1px),
  linear-gradient(to bottom, #ddd 1px, transparent 1px);
background-size: 20px 20px;

/* 渐变边框 */
.gradient-border {
  background:
    linear-gradient(white, white) padding-box,
    linear-gradient(to right, red, blue) border-box;
  border: 5px solid transparent;
}
```

## 四、精灵图（雪碧图）

### 4.1 什么是精灵图

**精灵图 = 把多张小图拼成一张大图，通过 background-position 显示不同部分**

```
┌─────────────────────────────────┐
│ 图1 │ 图2 │ 图3 │ 图4 │ 图5 │ 图6 │
└─────────────────────────────────┘
         一张大图

通过调整 position 显示：
图1: position: 0 0
图2: position: -50px 0
图3: position: -100px 0
```

### 4.2 精灵图使用

```css
.icon {
  width: 32px;
  height: 32px;
  background-image: url('sprite.png');
  background-repeat: no-repeat;
}

.icon-home {
  background-position: 0 0;
}

.icon-user {
  background-position: -32px 0;
}

.icon-settings {
  background-position: -64px 0;
}
```

### 4.3 精灵图的优势

| 优势 | 说明 |
|-----|------|
| 减少请求 | 1张图代替N张图 |
| 加载更快 | 减少HTTP请求开销 |
| 防止闪烁 | 预加载所有图标 |

## 五、响应式背景图

### 5.1 媒体查询切换

```css
.hero {
  background-image: url('mobile-bg.jpg');
}

@media (min-width: 768px) {
  .hero {
    background-image: url('desktop-bg.jpg');
  }
}
```

### 5.2 image-set（根据分辨率）

```css
.hero {
  background-image: image-set(
    'bg.jpg' 1x,
    'bg@2x.jpg' 2x,
    'bg@3x.jpg' 3x
  );
}
```

### 5.3 源图像选择

```css
.hero {
  background-image: url('small.jpg');
}

@supports (background-image: url('small.avif')) {
  .hero {
    background-image: url('small.avif');
  }
}

@supports (background-image: url('small.webp')) {
  .hero {
    background-image: url('small.webp');
  }
}
```

## 七、性能优化

### 7.1 图片格式选择

| 格式 | 优点 | 缺点 | 适用场景 |
|-----|------|-----|---------|
| JPG | 压缩率高 | 无透明 | 照片 |
| PNG | 支持透明 | 文件大 | 图标、Logo |
| WebP | 压缩率更高 | 兼容性 | 现代浏览器 |
| AVIF | 最小 | 兼容性差 | 前沿项目 |
| SVG | 无损缩放 | 复杂图形大 | 图标 |

### 7.2 懒加载

```javascript
// Intersection Observer 实现
const lazyImages = document.querySelectorAll('[data-bg]');

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const bg = entry.target;
      bg.style.backgroundImage = `url(${bg.dataset.bg})`;
      observer.unobserve(bg);
    }
  });
});

lazyImages.forEach(img => observer.observe(img));
```

```css
/* CSS 占位 */
.lazy-bg {
  background-color: #f0f0f0;
  min-height: 200px;
}
```

### 7.3 压缩优化

```css
/* 使用 CSS 渐变代替小图片 */
.gradient-bg {
  background: linear-gradient(to bottom, #3498db, #2980b9);
  /* 代替用一张渐变背景图 */
}

/* 使用 SVG 代替位图 */
.icon {
  background-image: url('icon.svg');
  /* 文件更小，可任意缩放 */
}
```
