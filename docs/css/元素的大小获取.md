# 元素的大小获取

## API 对比

| API | 宽度/高度 | 包含 | 场景 |
|-----|----------|------|------|
| `offsetWidth/Height` | 整数 | border + padding + content | 元素占位大小 |
| `clientWidth/Height` | 整数 | padding + content | 内容区域大小 |
| `scrollWidth/Height` | 整数 | 滚动内容的实际大小 | 滚动内容大小 |
| `getBoundingClientRect()` | 浮点数 | border + padding + content | 精确尺寸 + 位置 |
| `getComputedStyle()` | 字符串 | content | 样式声明的值 |

## 各 API 详解

### 1. offsetWidth / offsetHeight

元素的**占位大小**，包含 border、padding 和 content

```js
const box = document.querySelector('.box');

box.offsetWidth;   // border-left + padding-left + width + padding-right + border-right
box.offsetHeight;   // border-top + padding-top + height + padding-bottom + border-bottom
```

**模型：**
```
┌─────────────────────────────────┐
│           border                 │  ←
├─────────────────────────────────┤  │
│          padding                 │  │ offsetHeight
│   ┌─────────────────────┐       │  │
│   │     content         │       │  │
│   │                     │       │  │
│   └─────────────────────┘       │  │
│          padding                 │  │
├─────────────────────────────────┤  │
│           border                 │  ←
└─────────────────────────────────┘
```

### 2. clientWidth / clientHeight

元素的**内容区域大小**，包含 content 和 padding，**不含 border**

```js
box.clientWidth;   // padding-left + width + padding-right
box.clientHeight;  // padding-top + height + padding-bottom
```

**注意：** 如果元素有滚动条，clientWidth/Height 会减去滚动条的宽度（通常 15-17px）

### 3. scrollWidth / scrollHeight

元素的**滚动内容实际大小**，当内容超出时大于 clientWidth/Height

```js
// 内容无溢出时
box.scrollWidth === box.clientWidth;  // true

// 内容溢出时
box.scrollWidth > box.clientWidth;    // true
```

**模型：**
```
┌─────────────────────┐
│     可视区域         │ ← clientWidth
│   ┌───────────────┐ │
│   │   滚动内容     │ │
│   │               │ │
│   │               │ │ ← scrollWidth (更宽)
│   └───────────────┘ │
└─────────────────────┘
```

### 4. getBoundingClientRect()

返回元素的精确尺寸和位置，结果为浮点数

```js
const rect = box.getBoundingClientRect();

rect.width;   // 同 offsetWidth，但精确到小数
rect.height;  // 同 offsetHeight

// 同时返回位置
rect.top; rect.left; rect.bottom; rect.right;
```

### 5. getComputedStyle()

获取 CSS 样式中声明的尺寸值

```js
const style = window.getComputedStyle(box);

style.width;    // "100px" （字符串）
style.height;   // "200px"

style.boxSizing; // "border-box" 或 "content-box"
```

**注意：** 返回的是解析后的值，如 `100px`、`50%`、`10rem` 等

## box-sizing 的影响

### content-box（默认）

```css
.box {
  width: 100px;
  padding: 10px;
  border: 5px solid;
  box-sizing: content-box;
}
```

| 属性 | 值 | 说明 |
|------|-----|------|
| style.width | 100px | CSS 设置的宽度 |
| clientWidth | 120px | 100 + 10*2 |
| offsetWidth | 130px | 100 + 10*2 + 5*2 |

### border-box

```css
.box {
  width: 100px;
  padding: 10px;
  border: 5px solid;
  box-sizing: border-box;
}
```

| 属性 | 值 | 说明 |
|------|-----|------|
| style.width | 100px | CSS 设置的宽度 |
| clientWidth | 90px | 100 - 5*2 |
| offsetWidth | 100px | 等于设置的 width |

## 常见场景

### 1. 获取元素实际显示大小

```js
function getElementSize(element) {
  const rect = element.getBoundingClientRect();
  return {
    width: rect.width,
    height: rect.height,
    top: rect.top,
    left: rect.left
  };
}
```

### 2. 检测内容是否溢出

```js
function isOverflowing(element) {
  return (
    element.scrollHeight > element.clientHeight ||
    element.scrollWidth > element.clientWidth
  );
}

// 检测垂直溢出
function hasVerticalOverflow(element) {
  return element.scrollHeight > element.clientHeight;
}
```

### 3. 获取内容滚动距离

```js
// 最大可滚动距离
const maxScrollTop = element.scrollHeight - element.clientHeight;
const maxScrollLeft = element.scrollWidth - element.clientWidth;

// 滚动进度百分比
const scrollPercent = element.scrollTop / maxScrollTop;
```

### 4. 响应式检测（类似 CSS 媒体查询）

```js
function isMobile() {
  return window.innerWidth < 768;
}

// 监听窗口变化
window.addEventListener('resize', () => {
  if (isMobile()) {
    console.log('移动端');
  }
});
```

### 5. 等待图片加载完成获取真实尺寸

```js
function getImageSize(imgElement) {
  if (imgElement.complete) {
    return {
      width: imgElement.naturalWidth,
      height: imgElement.naturalHeight
    };
  }

  return new Promise((resolve) => {
    imgElement.onload = () => {
      resolve({
        width: imgElement.naturalWidth,
        height: imgElement.naturalHeight
      });
    };
  });
}
```

### 6. 计算滚动条宽度

```js
function getScrollbarWidth() {
  const outer = document.createElement('div');
  outer.style.cssText = 'position:absolute;top:0;left:0;width:100px;height:100px;overflow:scroll;visibility:hidden;';
  document.body.appendChild(outer);

  const inner = document.createElement('div');
  inner.style.cssText = 'width:100%;height:100%;';
  outer.appendChild(inner);

  const scrollbarWidth = outer.offsetWidth - inner.offsetWidth;

  outer.remove();
  return scrollbarWidth;
}

// 通常返回 15-17px
```

### 7. 监听元素尺寸变化

```js
const observer = new ResizeObserver((entries) => {
  for (const entry of entries) {
    console.log('尺寸变化:', entry.contentRect);
  }
});

observer.observe(element);

// 停止监听
// observer.unobserve(element);
// observer.disconnect();
```

## 注意事项

1. **offsetWidth/Height** 是整数，四舍五入
2. **getBoundingClientRect()** 返回浮点数，更精确
3. **display: none** 的元素，大小都为 0
4. **transform: scale()** 会影响 getBoundingClientRect()，不影响 offsetWidth
5. 读取这些属性会触发**回流**，频繁读取时注意性能
6. 宽度不含 **margin**，需要单独计算
