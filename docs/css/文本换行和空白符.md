### 1. word-wrap / overflow-wrap
```css
overflow-wrap: normal | break-word | anywhere;
```
#### 作用：处理长单词或URL的换行问题
- `normal`: 默认值，只在正常的单词断点处换行;
- `break-word`: 如果单词太长，在单词内部换行;
- `anywhere`: 在任意字符处换行，即使会断开单词；

### 2. word-break
```css
word-break: normal | break-all | keep-all | break-word;
```
#### 作用：控制非中文/日文/韩文（CJK）文本的换行方式
- normal：默认，CJK文本在字间换行，非CJK文本在单词间换行
- break-all：所有文本都在任意字符处换行，包括单词内部
- keep-all：CJK文本不换行，非CJK文本正常换行
- break-word：效果同 overflow-wrap: break-word（已弃用）

### 3. white-space
```css
white-space: normal | nowrap | pre | pre-wrap | pre-line;
```
#### 作用：控制空白符的处理方式
|值|空白符|换行符|自动换行|
|---|---|---|---|
|normal|合并|忽略|是|
|nowrap|合并|忽略|否|
|pre|保留|保留|否|
|pre-wrap|保留|保留|是|
|pre-line|合并|保留|是|


### 4. 相同点
1. 都是CSS文本属性，用于控制文本的显示方式
2. 都影响文本的**换行行为**
3. 都可以解决文本溢出容器的问题
4. 都可以应用在所有HTML元素上

### 5. 不同点
|属性|主要用途|处理对象|是否影响空白符|兼容性|
|---|---|---|---|---|
|overflow-wrap|处理长单词/URL|单词边界|否|CSS3|
|word-break|控制换行策略|字符级别|否|较好|
|white-space|处理空白符和换行|空白字符|是|好|
