# CSS 百分比完全指南

## 概述

CSS 中的百分比值总是相对于某个**参考系**进行计算的，这个参考系通常称为**包含块（Containing Block）**。不同属性的百分比值有不同的计算规则。

---

## 一、包含块（Containing Block）

包含块是 CSS 中定位和百分比计算的核心概念。一个元素的包含块取决于它的 `position` 属性：

### 1.1 包含块的确定规则

| position 值 | 包含块 |
|------------|--------|
| `static` / `relative` | 最近的块级祖先元素的 **content box**（内容区域） |
| `absolute` | 最近的 `position` 不为 `static` 的祖先元素的 **padding box**（内边距区域） |
| `fixed` | **视口**（viewport） |
| `sticky` | 滚动容器（最近的滚动祖先）的 **content box** |

### 1.2 Box 模型对照

```
┌─────────────────────────────────────┐
│           margin box                 │
│   ┌─────────────────────────────┐   │
│   │        border box           │   │
│   │   ┌─────────────────────┐   │   │
│   │   │    padding box      │   │   │
│   │   │  ┌───────────────┐  │   │   │
│   │   │  │  content box  │  │   │   │
│   │   │  │   (内容区域)   │  │   │   │
│   │   │  └───────────────┘  │   │   │
│   │   └─────────────────────┘   │   │
│   └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

## 二、常见属性的百分比计算规则

### 2.1 尺寸相关属性

#### `width` / `height`

| 属性 | 参考系 | 备注 |
|------|--------|------|
| `width` | 包含块的 **content box 宽度** | - |
| `height` | 包含块的 **content box 高度** | 父元素显式高度时才生效 |

```css
/* 示例：width 百分比 */
.parent { width: 600px; }
.child  { width: 50%; }  /* 实际宽度 = 300px */

/* 陷阱：height 百分比 */
.parent { height: auto; }  /* 未显式定义高度 */
.child  { height: 100%; }  /* ❌ 可能失效，实际高度由内容决定 */
```

<div style="background-color: #fff3cd; border-left: 4px solid #ffc107; padding: 10px; margin: 10px 0;">
<strong>⚠️ 重要提示：</strong>绝对定位元素的 <code>width</code>/<code>height</code> 也相对于包含块的 <strong>content box</strong>，而非 padding box。
</div>

---

### 2.2 内外边距属性

#### `padding` / `margin`

| 属性 | 参考系 | 备注 |
|------|--------|------|
| `padding-*` | 包含块的 **content box 宽度** | 即使是垂直方向 padding 也参考宽度 |
| `margin-*` | 包含块的 **content box 宽度** | 即使是垂直方向 margin 也参考宽度 |

```css
/* 示例：padding/margin 总是参考宽度 */
.parent { width: 600px; }
.child  {
  padding-top: 50%;   /* = 300px，参考宽度而非高度 */
  margin-left: 20%;   /* = 120px */
}
```

<div style="background-color: #d1ecf1; border-left: 4px solid #0dcaf0; padding: 10px; margin: 10px 0;">
<strong>💡 为什么垂直方向也参考宽度？</strong>这是 CSS 的历史设计决策，可以防止循环依赖：如果子元素的 padding 参考父元素高度，而父元素高度又由子元素内容决定，就会产生无限循环。
</div>

---

### 2.3 定位属性

#### `top` / `bottom` / `left` / `right`

| 属性 | 参考系 | 绝对定位元素参考系 |
|------|--------|-------------------|
| `left` / `right` | 包含块的 **content box 宽度** | 包含块的 **padding box 宽度** |
| `top` / `bottom` | 包含块的 **content box 高度** | 包含块的 **padding box 高度** |

```css
/* 示例：定位属性的百分比 */
.parent {
  width: 600px;
  height: 400px;
  padding: 20px;
  position: relative;
}

.child {
  position: absolute;
  top: 50%;      /* = 220px，参考 padding box 高度 (440px = 400 + 20×2) */
  left: 50%;     /* = 320px，参考 padding box 宽度 (640px = 600 + 20×2) */
}
```

<div style="background-color: #f8d7da; border-left: 4px solid #dc3545; padding: 10px; margin: 10px 0;">
<strong>🔴 绝对定位的特殊性：</strong>只有 <code>top</code>/<code>bottom</code>/<code>left</code>/<code>right</code> 是相对于 padding box 计算的，<code>width</code>/<code>height</code> 仍是相对于 content box。
</div>

---

### 2.4 字体相关属性

#### `font-size`

| 属性 | 参考系 |
|------|--------|
| `font-size` | **父元素**的 `font-size` 值 |

```css
/* 示例：font-size 百分比 */
html { font-size: 16px; }
body { font-size: 100%; }  /* = 16px */
h1   { font-size: 150%; }  /* = 24px */
```

#### `line-height`

| 属性 | 参考系 |
|------|--------|
| `line-height` | **元素自身**的 `font-size` 值 |

```css
/* 示例：line-height 百分比 */
.element {
  font-size: 16px;
  line-height: 150%;  /* = 24px，基于自身 font-size */
}
```

---

### 2.5 背景属性

#### `background-size` / `background-position`

| 属性 | 参考系 |
|------|--------|
| `background-size` | **背景定位区域**（默认为 padding box） |
| `background-position` | **背景定位区域** 减去 **背景图片尺寸** |

```css
/* 示例：background 相关 */
.element {
  width: 400px;
  height: 300px;
  background-size: 50% 50%;    /* 图片 = 200px × 150px */
  background-position: 50% 50%; /* 居中显示 */
}
```

---

### 2.6 圆角属性

#### `border-radius`

| 属性 | 参考系 |
|------|--------|
| `border-radius` | **元素自身**的 `width` 和 `height` |

```css
/* 示例：border-radius 百分比 */
.element {
  width: 200px;
  height: 100px;
  border-radius: 50%;  /* 水平半径=100px，垂直半径=50px */
}

/* 特殊效果：椭圆/圆形 */
.circle {
  width: 100px;
  height: 100px;
  border-radius: 50%;  /* 正圆形 */
}

.ellipse {
  width: 200px;
  height: 100px;
  border-radius: 50%;  /* 椭圆形 */
}
```

---

### 2.7 变换属性

#### `transform: translate()`

| 函数 | 参考系 |
|------|--------|
| `translateX(%)` | **元素自身**的宽度 |
| `translateY(%)` | **元素自身**的高度 |

```css
/* 示例：translate 百分比 */
.element {
  width: 200px;
  height: 100px;
  transform: translate(50%, 50%);  /* 向右100px，向下50px */
}

/* 常见应用：居中 */
.center {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);  /* 完美居中 */
}
```

---

### 2.8 其他属性

#### `flex` 子元素

| 属性 | 参考系 |
|------|--------|
| `flex-basis` | flex 容器的 **content box 宽度**（主轴为横向时） |

#### `grid` 子元素

| 属性 | 参考系 |
|------|--------|
| `grid-template-columns/rows` | grid 容器的 **content box** |

#### `object-fit`

| 属性 | 参考系 |
|------|--------|
| 不支持百分比 | 只能用关键字：`fill` / `contain` / `cover` / `none` / `scale-down` |

## 三、快速查询表

| 属性类别 | 属性 | 百分比参考系 |
|---------|------|-------------|
| 尺寸 | `width` | 包含块 content box 宽度 |
| 尺寸 | `height` | 包含块 content box 高度 |
| 间距 | `padding-*` / `margin-*` | 包含块 content box **宽度**（垂直方向也参考宽度） |
| 定位 | `top` / `bottom` | 包含块 content box 高度（绝对定位：padding box） |
| 定位 | `left` / `right` | 包含块 content box 宽度（绝对定位：padding box） |
| 字体 | `font-size` | 父元素的 font-size |
| 字体 | `line-height` | 元素自身的 font-size |
| 背景 | `background-size` | 背景定位区域 |
| 圆角 | `border-radius` | **元素自身**的 width/height |
| 变换 | `translateX/Y` | **元素自身**的 width/height |

## 四、常见陷阱

### 4.1 height: 100% 不生效

```css
/* ❌ 错误：父元素高度为 auto */
.parent { height: auto; }
.child  { height: 100%; }  /* 无效 */

/* ✅ 正确：父元素需要明确高度 */
.parent { height: 500px; }  /* 或 100vh / fixed 值 */
.child  { height: 100%; }  /* 有效 */
```

### 4.2 绝对定位的 width vs top/left

```css
.parent {
  width: 600px;
  padding: 50px;
  position: relative;
}

.child {
  position: absolute;
  width: 100%;    /* = 600px，参考 content box */
  left: 100%;     /* = 700px，参考 padding box (600+50+50) */
}
```

### 4.3 border-radius: 100% 的效果

```css
/* 宽高相等 → 圆形 */
.square { width: 100px; height: 100px; border-radius: 100%; }

/* 宽高不等 → 椭圆 */
.rect   { width: 200px; height: 100px; border-radius: 100%; }
```

## 五、实战技巧

### 5.1 完美居中方案

```css
/* 方案一：绝对定位 + translate */
.center-1 {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 方案二：flex */
.center-2 {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

### 5.2 响应式容器

```css
/* 保持固定宽高比 */
.container {
  width: 100%;
  aspect-ratio: 16 / 9;  /* 现代浏览器 */
  /* 或使用 padding hack（旧浏览器） */
}
```

### 5.3 使用 CSS 变量简化计算

```css
:root {
  --spacing-unit: 8px;
}

.element {
  padding: calc(var(--spacing-unit) * 2);  /* 16px */
  margin: calc(var(--spacing-unit) * 3);   /* 24px */
}
```
