### 一。包含块
<strong>包含块的确定规则</strong>
1. `position: static或relative`: 包含块是<strong>最近的块级祖先元素的内容区域(`width`)(`content box`)</strong>。如果没有块级祖先，则默认为根元素
2. `position:  absolute`: 包含块是其最近的`position`值不为`static`的祖先元素的<strong>内边距边界(`padding`)</strong>。如果没有这样的祖先元素，则包含块是根元素
3. `position: fixed`‌： 包含块是<strong>视口</strong>
4. <strong>普通元素：</strong> 包含块是<strong>最近的块级祖先元素的内容区域(`width`)(`content box`)</strong>

### 一。 百分比的参考系(包含块)
#### 百分比的值始终相对于其元素的包含块‌（Containing Block）‌
#### 绝对定位元素的百分比： 相对于包含块的<strong>内边距区域</strong>(`padding`)计算
1. **普通元素**：参考系为父元素的内容区域（content box）‌
2. **绝对定位**: 参考系为最近定位祖先元素的`padding`<strong>(内边距)</strong>区域.即父级元素设置了position: relative/absolute/fixed）‌
  * `position: static` （静态定位）或 `position: relative`（相对定位）: 参考系通常是由它最近的块级、表格单元格或行内块祖先元素的`内容区域（content box）`确定
  * `position: absolute`（绝对定位）： 其参考系由最近的、position值不为static（即relative、absolute或fixed）的祖先元素的<strong>内边距区域</strong>(`padding`)确定
  * `position: fixed`（固定定位）：相对于浏览器窗口进行定位


### 二。 常见属性的百分比计算规则

<div style="background-color:red;color:#fff">绝对定位元素的百分比： 相对于包含块的<strong>内边距区域</strong>(`padding`)计算</div>

1. `width` 和 `height`
* `width`：百分比相对于参考系的宽度‌(`wdith`)<strong>(content box 内容区域)</strong>
  * 示例：父元素宽600px，子元素设置`width: 50%`，则实际宽度为300px‌
* `height`：百分比相对于参考系的高度‌(`height`)<strong>(content box 内容区域)</strong>
  * 陷阱：若父元素高度未显式定义（如`height: auto`），子元素的`height: 100%`可能失效‌


2. `padding`、`margin`、`border`
<div style="background-color:red;color:#fff">绝对定位元素的百分比： 相对于包含块的<strong>内边距区域</strong>(`padding`)计算</div>

* 百分比均相对于参考系的‌<strong>宽度‌(width)</strong>计算‌


### 三。 `top`、`bottom`、`left`、`right`

<div style="background-color:red;color:#fff">绝对定位元素的百分比： 相对于包含块的<strong>内边距区域</strong>(`padding`)计算</div>

* `left`/`right`: 相对于参考系的<strong>宽度`(width)`</strong>
* `top`/`bottom`： 相对于参考系的<strong>高度`(height)`</strong>

### 四。`font-size`
* 相对于<strong>父元素的字体大小‌</strong>

### 五。`border-radius` 的百分比
* 相对于<strong>父元素</strong>的`width`和`height`。分别计算水平和垂直半径‌

### 六。`line-height`
* 相对于<strong>自身</strong>的`font-size`值‌

### 七。 `transform` 中的百分比
* `translateX(50%)`：相对于<strong>自身宽度</strong>的一半‌
* `translateY(50%)`：相对于<strong>自身高度</strong>的一半‌
