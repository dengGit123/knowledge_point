# 鼠标的位置

## 鼠标事件的位置属性

鼠标事件对象 `MouseEvent` 包含多个位置属性：

| 属性 | 参考系 | 说明 |
|------|--------|------|
| `clientX/Y` | 视口 | 相对浏览器可视区域 |
| `pageX/Y` | 文档 | 相对整个页面（含滚动） |
| `screenX/Y` | 屏幕 | 相对显示器屏幕 |
| `offsetX/Y` | 目标元素 | 相对事件目标元素 |
| `movementX/Y` | 上次位置 | 相对上一次鼠标移动的差值 |

## 各属性详解

### 1. clientX / clientY

相对**视口（viewport）**的位置，不受滚动影响

```js
document.addEventListener('mousemove', (e) => {
  console.log(e.clientX, e.clientY);
});
```

**模型：**
```
┌─────────────────────────────┐
│ 视口 (0,0)                  │
│                             │
│     ● 鼠标                  │  ← clientX/Y
│     (clientX, clientY)      │
│                             │
└─────────────────────────────┘
    ↓ 滚动 ↓
┌─────────────────────────────┐
│     页面下方内容            │
└─────────────────────────────┘
```

### 2. pageX / pageY

相对**整个文档**的位置，包含滚动的距离

```js
document.addEventListener('mousemove', (e) => {
  console.log(e.pageX, e.pageY);
});

// 关系：
// pageX = clientX + window.scrollX
// pageY = clientY + window.scrollY
```

### 3. screenX / screenY

相对**显示器屏幕**的位置，不受浏览器窗口位置影响

```js
document.addEventListener('mousemove', (e) => {
  console.log(e.screenX, e.screenY);
});
```

**使用场景：** 多显示器相关操作、打开新窗口定位

### 4. offsetX / offsetY

相对**事件目标元素**左上角的位置

```js
element.addEventListener('click', (e) => {
  console.log(e.offsetX, e.offsetY);
  // 点击位置相对于 element 内部的坐标
});
```

**注意：** 如果子元素触发事件，坐标是相对于子元素的！这是最大的坑：

```html
<!-- ❌ 问题场景 -->
<div class="box">
  <span>文字</span>  <!-- 点击这里，offsetX 是相对于 span 的！
</div>
```

### 5. movementX / movementY

相对**上一次 mousemove 事件**的位移

```js
element.addEventListener('mousemove', (e) => {
  console.log(e.movementX, e.movementY);
  // 正值 = 向右/向下移动
  // 负值 = 向左/向上移动
});
```

**注意：** 只在 `mousemove` 事件中有值

## 常见场景

### 1. 跟随鼠标移动

```js
const follower = document.querySelector('.follower');

document.addEventListener('mousemove', (e) => {
  follower.style.left = `${e.clientX}px`;
  follower.style.top = `${e.clientY}px`;
});
```

### 2. 获取鼠标相对于元素的位置

```js
element.addEventListener('click', (e) => {
  const rect = element.getBoundingClientRect();
  const x = e.clientX - rect.left;
  const y = e.clientY - rect.top;

  console.log(`点击位置：(${x}, ${y})`);
});
```

### 3. 拖拽元素

```js
let isDragging = false;
let startX, startY, initialLeft, initialTop;

draggable.addEventListener('mousedown', (e) => {
  isDragging = true;
  startX = e.clientX;
  startY = e.clientY;

  const rect = draggable.getBoundingClientRect();
  initialLeft = rect.left;
  initialTop = rect.top;
});

document.addEventListener('mousemove', (e) => {
  if (!isDragging) return;

  const dx = e.clientX - startX;
  const dy = e.clientY - startY;

  draggable.style.position = 'fixed';
  draggable.style.left = `${initialLeft + dx}px`;
  draggable.style.top = `${initialTop + dy}px`;
});

document.addEventListener('mouseup', () => {
  isDragging = false;
});
```

### 4. 鼠标悬停检测

```js
function isMouseOver(element) {
  const rect = element.getBoundingClientRect();
  return (
    mouseX >= rect.left &&
    mouseX <= rect.right &&
    mouseY >= rect.top &&
    mouseY <= rect.bottom
  );
}

let mouseX, mouseY;
document.addEventListener('mousemove', (e) => {
  mouseX = e.clientX;
  mouseY = e.clientY;
});
```

### 5. 画板/绘图应用

```js
const canvas = document.querySelector('canvas');
const ctx = canvas.getContext('2d');
let isDrawing = false;

function getCanvasPosition(e) {
  const rect = canvas.getBoundingClientRect();
  return {
    x: e.clientX - rect.left,
    y: e.clientY - rect.top
  };
}

canvas.addEventListener('mousedown', (e) => {
  isDrawing = true;
  const pos = getCanvasPosition(e);
  ctx.beginPath();
  ctx.moveTo(pos.x, pos.y);
});

canvas.addEventListener('mousemove', (e) => {
  if (!isDrawing) return;
  const pos = getCanvasPosition(e);
  ctx.lineTo(pos.x, pos.y);
  ctx.stroke();
});

document.addEventListener('mouseup', () => {
  isDrawing = false;
});
```

### 6. 鼠标移入方向判断

```js
function getDirection(element, e) {
  const rect = element.getBoundingClientRect();
  const x = e.clientX - rect.left - rect.width / 2;
  const y = e.clientY - rect.top - rect.height / 2;

  return Math.abs(x) > Math.abs(y)
    ? x > 0 ? 'right' : 'left'
    : y > 0 ? 'bottom' : 'top';
}

element.addEventListener('mouseenter', (e) => {
  const direction = getDirection(element, e);
  console.log(`从${direction}移入`);
});
```

### 7. 鼠标速度计算

```js
let lastTime = Date.now();
let lastX = 0, lastY = 0;

document.addEventListener('mousemove', (e) => {
  const now = Date.now();
  const dt = now - lastTime;

  if (dt > 0) {
    const dx = e.clientX - lastX;
    const dy = e.clientY - lastY;
    const distance = Math.sqrt(dx * dx + dy * dy);
    const speed = distance / dt; // 像素/毫秒

    console.log(`鼠标速度：${speed.toFixed(2)} px/ms`);
  }

  lastTime = now;
  lastX = e.clientX;
  lastY = e.clientY;
});
```

### 8. 自定义右键菜单

```js
const contextMenu = document.querySelector('.context-menu');

document.addEventListener('contextmenu', (e) => {
  e.preventDefault();

  // 确保菜单不会超出视口
  const x = Math.min(e.clientX, window.innerWidth - 200);
  const y = Math.min(e.clientY, window.innerHeight - 150);

  contextMenu.style.left = `${x}px`;
  contextMenu.style.top = `${y}px`;
  contextMenu.classList.add('visible');
});

document.addEventListener('click', () => {
  contextMenu.classList.remove('visible');
});
```

### 9. 鼠标悬停放大效果

```js
container.addEventListener('mousemove', (e) => {
  const rect = container.getBoundingClientRect();
  const x = (e.clientX - rect.left) / rect.width;
  const y = (e.clientY - rect.top) / rect.height;

  // 根据鼠标位置计算缩放原点
  image.style.transformOrigin = `${x * 100}% ${y * 100}%`;
  image.style.transform = 'scale(2)';
});

container.addEventListener('mouseleave', () => {
  image.style.transform = 'scale(1)';
});
```

### 10. 惯性滚动效果

```js
let mouseX = 0, mouseY = 0;
let currentX = 0, currentY = 0;

document.addEventListener('mousemove', (e) => {
  mouseX = e.clientX;
  mouseY = e.clientY;
});

function animate() {
  // 缓动跟随
  currentX += (mouseX - currentX) * 0.1;
  currentY += (mouseY - currentY) * 0.1;

  follower.style.left = `${currentX}px`;
  follower.style.top = `${currentY}px`;

  requestAnimationFrame(animate);
}

animate();
```

## 鼠标在元素内的位置

这是一个非常常见的需求，比如点击画布绘图、图片局部放大等。

### 方法对比

| 方法 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| `offsetX/Y` | 简单直接 | 子元素触发时坐标错误 | ⭐⭐ |
| `getBoundingClientRect()` | 可靠、不受子元素影响 | 需要手动计算 | ⭐⭐⭐ |
| 封装工具函数 | 功能完整、可复用 | 需要额外代码 | ⭐⭐⭐ |

### 方法一：使用 `offsetX/Y`（简单场景）

```js
element.addEventListener('click', (e) => {
  console.log(e.offsetX, e.offsetY);
});
```

**适用场景：** 元素内没有子元素，或子元素设置了 `pointer-events: none;`

```css
.child {
  pointer-events: none;  /* 让点击穿透到父元素 */
}
```

### 方法二：使用 `getBoundingClientRect()`（推荐）

```js
element.addEventListener('click', (e) => {
  const rect = element.getBoundingClientRect();
  const x = e.clientX - rect.left;
  const y = e.clientY - rect.top;
  console.log(x, y);
});
```

**为什么推荐：**
- 坐标始终相对于绑定事件的元素
- 不受子元素影响
- 计算简单直观

### 方法三：封装工具函数

```js
/**
 * 获取鼠标在元素内的位置
 * @param {HTMLElement} element - 目标元素
 * @param {MouseEvent} event - 鼠标事件
 * @returns {{x: number, y: number, normalizedX: number, normalizedY: number}}
 */
function getMouseInElement(element, event) {
  const rect = element.getBoundingClientRect();
  const x = event.clientX - rect.left;
  const y = event.clientY - rect.top;

  return {
    x,                          // 像素坐标
    y,
    normalizedX: x / rect.width,   // 归一化坐标 0-1
    normalizedY: y / rect.height,
    percentX: Math.round((x / rect.width) * 100),  // 百分比
    percentY: Math.round((y / rect.height) * 100)
  };
}

// 使用
element.addEventListener('click', (e) => {
  const pos = getMouseInElement(element, e);
  console.log(pos);
  // { x: 150, y: 75, normalizedX: 0.5, normalizedY: 0.5, percentX: 50, percentY: 50 }
});
```

### 实际应用：点击图片获取颜色

```js
const img = document.querySelector('img');
const canvas = document.createElement('canvas');
const ctx = canvas.getContext('2d');

img.addEventListener('click', (e) => {
  const rect = img.getBoundingClientRect();
  const x = e.clientX - rect.left;
  const y = e.clientY - rect.top;

  // 获取图片在点击位置的颜色
  canvas.width = img.naturalWidth;
  canvas.height = img.naturalHeight;
  ctx.drawImage(img, 0, 0);

  // 转换坐标到原图尺寸
  const scaleX = img.naturalWidth / rect.width;
  const scaleY = img.naturalHeight / rect.height;
  const pixel = ctx.getImageData(x * scaleX, y * scaleY, 1, 1).data;

  console.log(`RGB: rgb(${pixel[0]}, ${pixel[1]}, ${pixel[2]})`);
});
```

### 实际应用：在指定位置创建元素

```js
container.addEventListener('dblclick', (e) => {

  tooltip.textContent = '双击位置';
  tooltip.style.position = 'absolute';
  tooltip.style.left = `${pos.x}px`;
  tooltip.style.top = `${pos.y}px`;
  tooltip.style.transform = 'translate(-50%, -100%)';

  container.appendChild(tooltip);
});
```

## 触摸设备兼容

### 触摸事件位置属性对照

| 触摸事件属性 | 对应鼠标事件属性 |
|-------------|-----------------|
| `touches[0].clientX` | `clientX` |
| `touches[0].pageX` | `pageX` |
| `touches[0].screenX` | `screenX` |

### 通用位置获取函数

```js
function getEventPosition(e) {
  if (e.touches && e.touches.length > 0) {
    return { x: e.touches[0].clientX, y: e.touches[0].clientY };
  }
  return { x: e.clientX, y: e.clientY };
}
```

### 在元素内位置（兼容触摸）

```js
function getMouseInElement(element, event) {
  const rect = element.getBoundingClientRect();

  // 兼容触摸和鼠标
  const clientX = event.touches
    ? event.touches[0].clientX
    : event.clientX;
  const clientY = event.touches
    ? event.touches[0].clientY
    : event.clientY;

  return {
    x: clientX - rect.left,
    y: clientY - rect.top
  };
}

// 使用
element.addEventListener('click', handleTap);
element.addEventListener('touchstart', handleTap);

function handleTap(e) {
  // 防止触摸同时触发 click
  if (e.type === 'touchstart') {
    e.preventDefault(); // 可选
  }
  const pos = getMouseInElement(element, e);
  console.log(pos);
}
```

## 注意事项

### 鼠标在元素内位置
1. **优先用 `getBoundingClientRect()`**，不用 `offsetX/Y`，避免子元素问题
2. 元素有 `transform` 时，坐标计算仍然准确
3. 元素有 `border` 时，`getBoundingClientRect()` 包含 border
4. 想要不含 border 的坐标，需减去 `border-left` 和 `border-top`

### 通用
1. **`clientX/Y`** 最常用，适合大多数场景
2. **`offsetX/Y`** 注意事件冒泡，可能是子元素触发
3. **`movementX/Y`** 只在 `mousemove` 中有效
4. 移动端使用 `touch` 事件，不是 `mouse` 事件
5. 频繁的 `mousemove` 可能影响性能，考虑节流或 `requestAnimationFrame`

### 性能优化示例

```js
// ❌ 频繁触发，性能差
element.addEventListener('mousemove', (e) => {
  const pos = getMouseInElement(element, e);
  updateUI(pos);  // 每次移动都更新 UI
});

// ✅ 使用 RAF 优化
let rafId = null;
element.addEventListener('mousemove', (e) => {
  if (rafId) return;
  rafId = requestAnimationFrame(() => {
    const pos = getMouseInElement(element, e);
    updateUI(pos);
    rafId = null;
  });
});
```
