## 什么是伪元素？

**伪元素（Pseudo-element）** 是 CSS 中用于**创建文档中不存在的虚拟元素**，并对其进行样式化的机制。

想象一下你在写文章，想给第一段的第一个字加特殊样式，或者在每个链接后面加个小图标：

```
┌─────────────────────────────────────────────────────┐
│  首字下沉效果                                       │
│                                                     │
│  【我】是一段文字的开头，第一个字需要特殊样式。        │
│  这是后面的文字，保持正常样式。                       │
│                                                     │
└─────────────────────────────────────────────────────┘

链接后面加图标：
点击这里 [🔗]
```

伪元素就是用来实现这种"凭空创造"的元素样式效果。

## 一、伪元素的语法

### 单冒号 vs 双冒号

```css
/* CSS2 语法（单冒号）- 兼容旧浏览器 */
p:before { content: ""; }
p:first-line { color: red; }

/* CSS3 语法（双冒号）- 推荐使用 */
p::before { content: ""; }
p::first-line { color: red; }
```

| 语法 | 适用范围 | 兼容性 | 推荐度 |
|-----|---------|-------|-------|
| `::` | CSS3 新伪元素 | IE9+ | ⭐⭐⭐⭐⭐ |
| `:` | CSS2 伪元素兼容 | IE6+ | ⭐⭐⭐ |

> 💡 **最佳实践**：使用双冒号 `::` 来区分伪元素和伪类

## 二、常用伪元素详解

### 2.1 `::before` - 在元素内容之前插入

**作用**：在目标元素的**内容前面**插入一个虚拟元素。

```css
/* 基础用法 */
p::before {
  content: "📝 ";  /* 必须有 content 属性 */
  color: blue;
}
```

```html
<p>这是一段文字</p>
<!-- 渲染结果：📝 这是一段文字 -->
```

**content 属性的多种值：**

| 值类型 | 示例 | 效果 |
|-------|------|------|
| 字符串 | `content: "★";` | 插入文字/符号 |
| 空字符串 | `content: "";` | 创建纯样式元素 |
| URL | `content: url(icon.png);` | 插入图片 |
| attr() | `content: attr(data-text);` | 获取元素属性值 |
| counter() | `content: counter(num);` | 计数器 |

**实用示例：添加引号**

```css
.quote::before {
  content: "「";
  font-size: 2em;
  color: #999;
}

.quote::after {
  content: "」";
  font-size: 2em;
  color: #999;
}
```

```html
<p class="quote">这是一段引用</p>
<!-- 渲染结果：「这是一段引用」 -->
```

---

### 2.2 `::after` - 在元素内容之后插入

**作用**：在目标元素的**内容后面**插入一个虚拟元素。

**经典应用：清除浮动（Clearfix）**

```css
.clearfix::after {
  content: "";      /* 必须有 */
  display: table;   /* 触发块级格式化 */
  clear: both;      /* 清除左右浮动 */
}
```

```html
<div class="clearfix">
  <div style="float: left;">左浮动</div>
  <div style="float: right;">右浮动</div>
</div>
<!-- 父元素高度不会坍塌！ -->
```

**进度条实现：**

```css
.progress-bar {
  width: 200px;
  height: 20px;
  background: #eee;
  border-radius: 10px;
  position: relative;
}

.progress-bar::after {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  width: 60%;   /* 进度值 */
  height: 100%;
  background: linear-gradient(90deg, #4CAF50, #8BC34A);
  border-radius: 10px;
}
```

```
┌─────────────────────────────────────┐
│ ████████████████████░░░░░░░░░░░░░░░ │  ← 60% 进度
└─────────────────────────────────────┘
```

---

### 2.3 `::first-line` - 首行文本样式

**作用**：选中元素的**第一行文本**。

```css
p::first-line {
  font-weight: bold;
  font-size: 1.2em;
  color: #333;
}
```

```
┌─────────────────────────────────────┐
│ 【这是第一行文本，加粗显示】        │  ← 特殊样式
│ 这是第二行，保持正常样式。           │  ← 普通样式
│ 这是第三行，也是普通样式。           │
└─────────────────────────────────────┘
```

**注意事项：**

| 特性 | 说明 |
|-----|------|
| 只对块级元素生效 | `display: block/inline-block` 等 |
| 受容器宽度影响 | 宽度变化会改变"首行"范围 |
| 可继承属性 | 会继承父元素的某些属性 |

---

### 2.4 `::first-letter` - 首字样式

**作用**：选中元素的**第一个字符**。

**经典应用：首字下沉**

```css
p::first-letter {
  font-size: 3em;
  font-weight: bold;
  color: #E91E63;
  float: left;           /* 浮动让文字环绕 */
  margin-right: 8px;     /* 与后面文字的间距 */
  line-height: 1;        /* 避免行高过大 */
}
```

```
┌─────────────────────────────────────┐
│ 【我】是一段文字的开头，            │
│ 第一个字被放大显示。                │
│ 这是后面的文字...                   │
└─────────────────────────────────────┘
```

**注意事项：**

| 特性 | 说明 |
|-----|------|
| 只对块级元素生效 | 行内元素不支持 |
| 忽略前导空白 | 空格、制表符不被选中 |
| 标点符号也会被选中 | 如果首字符是标点，会一起选中 |

---

### 2.5 `::selection` - 选中文字样式

**作用**：自定义用户**选中文字时**的样式。

```css
::selection {
  background: #FFEB3B;  /* 选中背景色 */
  color: #333;           /* 选中文字颜色 */
}

/* 取消选中样式 */
.no-select::selection {
  background: transparent;
}
```

```
┌─────────────────────────────────────┐
│ 用户用鼠标选中这段文字时，          │
│ 背景变成黄色，文字变成深色。        │
└─────────────────────────────────────┘
```

**注意事项：**

| 属性 | 是否支持 |
|-----|---------|
| `color` | ✅ |
| `background` | ✅ |
| `cursor` | ✅ |
| `outline` | ✅ |
| `font-size` | ❌ |
| `text-transform` | ❌ |

---

### 2.6 `::placeholder` - 占位符样式

**作用**：自定义表单输入框的**占位符文字**样式。

```css
input::placeholder {
  color: #999;
  font-style: italic;
  font-size: 14px;
}
```

```html
<input type="text" placeholder="请输入用户名" />
<!-- 占位符显示灰色斜体文字 -->
```

**浏览器兼容性：**

```css
/* 完整兼容写法 */
input::-webkit-input-placeholder { /* Chrome/Safari */
  color: #999;
}
input::-moz-placeholder { /* Firefox 19+ */
  color: #999;
}
input:-ms-input-placeholder { /* IE 10+ */
  color: #999;
}
```

---

### 2.7 `::marker` - 列表项标记样式

**作用**：自定义列表（`<ul>`、`<ol>`、`<li>`）的**项目符号**或**编号**样式。

```css
li::marker {
  color: #4CAF50;
  font-size: 1.2em;
  content: "✓ ";   /* 自定义标记内容 */
}
```

```html
<ul>
  <li>第一项</li>
  <li>第二项</li>
</ul>
<!-- 渲染结果：✓ 第一项，✓ 第二项 -->
```

**实用示例：自定义数字列表**

```css
li::marker {
  content: "第" counter(list-item) "章 ";
  font-weight: bold;
  color: #607D8B;
}
```

```
第1章 简介
第2章 核心概念
第3章 实践指南
```

---

### 2.8 `::backdrop` - 全屏背景层

**作用**：在**全屏元素**（如 `<dialog>`、`<video>` 全屏模式）后面创建一个背景层。

```css
/* dialog 打开时的背景 */
dialog::backdrop {
  background: rgba(0, 0, 0, 0.7);
  backdrop-filter: blur(5px);  /* 毛玻璃效果 */
}

/* video 全屏时的背景 */
video::backdrop {
  background: #000;
}
```

```html
<dialog open>
  <p>这是一个对话框</p>
</dialog>
<!-- 背景会变暗模糊 -->
```

---

### 2.9 `::file-selector-button` - 文件选择按钮

**作用**：自定义 `<input type="file">` 的**选择按钮**样式。

```css
input[type="file"]::file-selector-button {
  padding: 8px 16px;
  background: #2196F3;
  color: white;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  transition: background 0.3s;
}

input[type="file"]::file-selector-button:hover {
  background: #1976D2;
}
```

```html
<input type="file" />
<!-- 按钮变成蓝色圆角样式 -->
```

---

### 2.10 `::grammar-error` 和 `::spelling-error` - 拼写/语法错误

**作用**：自定义浏览器标记的**拼写错误**或**语法错误**文字样式。

```css
/* 拼写错误（红色波浪线）*/
::spelling-error {
  text-decoration: underline wavy red;
}

/* 语法错误（绿色波浪线）*/
::grammar-error {
  text-decoration: underline wavy green;
}
```

## 三、伪元素的特性总结

### 3.1 伪元素的重要特性

| 特性 | 说明 |
|-----|------|
| **必须有 content** | `::before` 和 `::after` 必须设置 `content` 属性 |
| **行内元素** | 默认是 `display: inline`，需要宽高时改为 `block` 或 `inline-block` |
| **不能绑定事件** | 伪元素无法响应 JavaScript 事件（除非事件冒泡） |
| **可以使用伪类** | `a::before:hover` 不支持，但 `a:hover::before` 支持 |
| **CSS 计数器** | 配合 `counter-reset` 和 `counter-increment` 使用 |

### 3.2 伪元素 vs 伪类

| 对比项 | 伪元素 `::` | 伪类 `:` |
|-------|------------|---------|
| 作用 | 创建虚拟元素 | 选择元素状态 |
| 语法 | `::before`、`::after` | `:hover`、`:active` |
| 数量 | 一个选择器只能用一个伪元素 | 可多个伪类组合 |
| DOM 位置 | 元素内容内部 | 元素本身 |

```css
/* 伪元素：创建新元素 */
p::before { content: "★"; }

/* 伪类：选择状态 */
p:hover { color: red; }

/* 组合使用 */
a:hover::after { content: " →"; }
```

## 四、高级应用技巧

### 4.1 CSS 计数器

```css
/* 重置计数器 */
.list {
  counter-reset: item;
}

/* 递增计数器 */
.list li::before {
  counter-increment: item;
  content: counter(item) ". ";
  font-weight: bold;
}
```

```html
<ul class="list">
  <li>第一项</li>
  <li>第二项</li>
  <li>第三项</li>
</ul>
<!-- 渲染结果：
1. 第一项
2. 第二项
3. 第三项
-->
```

### 4.2 纯 CSS 图标

```css
.checkmark::before {
  content: "";
  display: inline-block;
  width: 20px;
  height: 20px;
  border: 2px solid #4CAF50;
  border-radius: 50%;
  background: #4CAF50;
  clip-path: polygon(20% 60%, 40% 80%, 80% 40%);
}
```

### 4.3 工具提示（Tooltip）

```css
.tooltip {
  position: relative;
  cursor: help;
}

.tooltip::before {
  content: attr(data-tooltip);
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  padding: 8px 12px;
  background: #333;
  color: white;
  border-radius: 4px;
  white-space: nowrap;
  opacity: 0;
  transition: opacity 0.3s;
  pointer-events: none;  /* 不阻挡鼠标事件 */
}

.tooltip:hover::before {
  opacity: 1;
}
```

```html
<span class="tooltip" data-tooltip="这是一个提示">悬停显示</span>
```

### 4.4 纯 CSS 三角形

```css
.triangle::after {
  content: "";
  width: 0;
  height: 0;
  border-left: 10px solid transparent;
  border-right: 10px solid transparent;
  border-bottom: 10px solid #333;
}
```

```
    ▲
   / \
  -----  ← 纯 CSS 绘制的三角形
```

## 五、常见问题与解决方案

### 问题1：伪元素无法显示

```css
/* ❌ 错误：缺少 content */
.element::before {
  width: 10px;
  height: 10px;
  background: red;
}

/* ✅ 正确：必须有 content */
.element::before {
  content: "";   /* 关键！ */
  width: 10px;
  height: 10px;
  background: red;
}
```

### 问题2：伪元素无法设置宽高

```css
/* ❌ 错误：默认是行内元素 */
.element::before {
  content: "";
  width: 100px;  /* 无效 */
  height: 100px; /* 无效 */
}

/* ✅ 正确：改为块级或行内块 */
.element::before {
  content: "";
  display: block;       /* 或 inline-block */
  width: 100px;
  height: 100px;
}
```

### 问题3：伪元素遮挡了父元素内容

```css
/* ❌ 伪元素覆盖了父元素文字 */
.element::before {
  content: "";
  position: absolute;
  width: 100%;
  height: 100%;
  background: rgba(0,0,0,0.5);
}

/* ✅ 调整层级 */
.element {
  position: relative;
}

.element::before {
  content: "";
  position: absolute;
  z-index: -1;   /* 放到后面 */
  width: 100%;
  height: 100%;
  background: rgba(0,0,0,0.5);
}
```

## 八、常见面试题

### Q1：伪元素和伪类的区别？

**答：**
- 伪类（`:`）：选择元素的某种状态，如 `:hover`、`:active`
- 伪元素（`::`）：创建文档中不存在的虚拟元素，如 `::before`、`::after`

### Q2：`::before` 和 `::after` 的 content 属性可以省略吗？

**答：** 不可以，必须设置 `content` 属性才能显示伪元素，即使是空字符串 `content: ""`。

### Q3：如何用伪元素实现清除浮动？

**答：** 使用 `::after` 伪元素，设置 `content: ""`、`display: table`、`clear: both`。

### Q4：伪元素可以绑定 JavaScript 事件吗？

**答：** 不能直接绑定，但事件会冒泡到父元素，可以在父元素上监听。

### Q5：`::first-letter` 对哪些元素生效？

**答：** 只对块级元素生效，行内元素需要先设置 `display: block`。

## 九、总结

### 核心要点

```
伪元素 = 虚拟元素 = 文档中不存在但可样式化的元素

常用伪元素：
┌─────────────────────────────────────────┐
│ ::before   → 内容前插入                 │
│ ::after    → 内容后插入（清除浮动）      │
│ ::first-line  → 首行样式                │
│ ::first-letter → 首字样式（首字下沉）    │
│ ::selection  → 选中文字样式             │
│ ::placeholder → 占位符样式              │
│ ::marker     → 列表标记样式             │
└─────────────────────────────────────────┘
```

### 最佳实践

| 场景 | 推荐伪元素 |
|-----|-----------|
| 清除浮动 | `::after` |
| 添加装饰 | `::before` / `::after` |
| 首字效果 | `::first-letter` |
| 首行效果 | `::first-line` |
| 表单美化 | `::placeholder` / `::file-selector-button` |
| 列表自定义 | `::marker` |

> 💡 **记住**：伪元素是 CSS 强大的"魔法"，能用纯 CSS 实现很多复杂效果，减少 DOM 操作！
