# CSS 媒体查询

> 官方文档：[MDN - @media](https://developer.mozilla.org/zh-CN/docs/Web/CSS/@media)

## 什么是媒体查询？

**媒体查询 = 让网页根据不同设备自动调整样式**

想象一下：
- 📱 手机屏幕小，内容要竖着排
- 💻 电脑屏幕大，内容可以横着排
- 🖨️ 打印时隐藏导航栏，只保留正文

媒体查询就是让**同一套 HTML** 在不同设备上显示**不同样式**的技术。

```
┌─────────────────────────────────────────────────┐
│                  HTML 结构（不变）               │
│                                                 │
│   ┌─────────┐      ┌───────────────┐           │
│   │  CSS    │ ───→ │ 媒体查询规则   │           │
│   └─────────┘      └───────────────┘           │
│                            ↓                    │
│   ┌─────────────────────────────────────────┐  │
│   │              不同设备不同样式              │  │
│   │  ┌───────┐  ┌────────┐  ┌─────────┐     │  │
│   │  │ 手机  │  │ 平板   │  │ 电脑    │     │  │
│   │  │ 样式  │  │ 样式   │  │ 样式    │     │  │
│   │  └───────┘  └────────┘  └─────────┘     │  │
│   └─────────────────────────────────────────┘  │
└─────────────────────────────────────────────────┘
```

## 一、基本语法

### 1.1 简写公式

```css
@media 媒体类型 and (媒体特性) {
  /* 满足条件时应用的样式 */
}
```

### 1.2 最简单的例子

```css
/* 屏幕宽度小于 768px 时（手机）*/
@media (max-width: 767px) {
  body {
    font-size: 14px;
  }
}

/* 屏幕宽度大于等于 768px 时（平板/电脑）*/
@media (min-width: 768px) {
  body {
    font-size: 16px;
  }
}
```

---

## 二、媒体类型

媒体类型描述设备的类别，常用的是 `screen`（屏幕）。

| 类型 | 说明 | 适用场景 |
|-----|------|---------|
| `all` | 所有设备（默认） | 通用 |
| `screen` | 屏幕设备 | 电脑、手机、平板 |
| `print` | 打印预览 | 打印样式 |
| `speech` | 语音合成 | 屏幕阅读器 |

```css
/* 屏幕设备 */
@media screen {
  body {
    font-size: 16px;
  }
}

/* 打印时 */
@media print {
  .nav, .footer {
    display: none;  /* 隐藏导航和页脚 */
  }
}

/* 所有设备（可省略不写）*/
@media all {
  body {
    margin: 0;
  }
}
```

> 💡 **现代写法通常省略媒体类型**，直接写条件即可：
> ```css
> @media (max-width: 768px) { ... }
> ```

## 三、媒体特性

媒体特性用来描述设备的具体特征，比如宽度、高度、分辨率等。

### 3.1 宽度相关（最常用）

| 特性 | 说明 | 示例 |
|-----|------|-----|
| `width` | 视口宽度 | `@media (width: 768px)` |
| `min-width` | 最小宽度（≥） | `@media (min-width: 768px)` |
| `max-width` | 最大宽度（≤） | `@media (max-width: 767px)` |

```css
/* 宽度刚好是 768px */
@media (width: 768px) {
  /* 很少单独使用 */
}

/* 宽度大于等于 768px */
@media (min-width: 768px) {
  /* 平板和电脑 */
}

/* 宽度小于等于 767px */
@media (max-width: 767px) {
  /* 手机 */
}

/* 常用：范围写法 */
@media (min-width: 768px) and (max-width: 1023px) {
  /* 只有平板 */
}
```

### 3.2 高度相关

| 特性 | 说明 | 示例 |
|-----|------|-----|
| `height` | 视口高度 | `@media (height: 600px)` |
| `min-height` | 最小高度 | `@media (min-height: 600px)` |
| `max-height` | 最大高度 | `@media (max-height: 600px)` |

```css
/* 高度较小的设备（横屏手机）*/
@media (max-height: 500px) {
  .header {
    padding: 10px;  /* 减小内边距 */
  }
}
```

### 3.3 方向

| 特性 | 值 | 说明 |
|-----|-----|-----|
| `orientation` | `portrait` | 竖屏（高度 > 宽度） |
| `orientation` | `landscape` | 横屏（宽度 > 高度） |

```css
/* 竖屏 */
@media (orientation: portrait) {
  .nav {
    flex-direction: column;  /* 竖着排 */
  }
}

/* 横屏 */
@media (orientation: landscape) {
  .nav {
    flex-direction: row;  /* 横着排 */
  }
}
```

### 3.4 分辨率（高清屏）

| 特性 | 说明 | 示例 |
|-----|------|-----|
| `resolution` | 设备分辨率 | `@media (resolution: 2dppx)` |
| `min-resolution` | 最小分辨率 | `@media (min-resolution: 2dppx)` |

```css
/* 高清屏（Retina显示屏等）*/
@media (min-resolution: 2dppx) {
  .logo {
    background-image: url('logo@2x.png');  /* 用高清图片 */
  }
}
```

> 💡 **dppx** = dots per pixel（每像素点数），2dppx = 2x 屏幕即常说的 Retina 屏

### 3.5 颜色相关

| 特性 | 说明 |
|-----|------|
| `color` | 设备颜色位数（0=黑白） |
| `color-gamut` | 色域范围（srgb, p3, rec2020） |
| `prefers-color-scheme` | 用户颜色偏好（light/dark） |

```css
/* 深色模式 */
@media (prefers-color-scheme: dark) {
  body {
    background: #1a1a1a;
    color: #ffffff;
  }
}

/* 浅色模式 */
@media (prefers-color-scheme: light) {
  body {
    background: #ffffff;
    color: #000000;
  }
}
```

### 3.6 用户偏好

| 特性 | 说明 |
|-----|------|
| `prefers-reduced-motion` | 减少动画（no-preference / reduce） |
| `prefers-contrast` | 对比度偏好（more / less / no-preference） |

```css
/* 用户开启了减少动画 */
@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}
```

## 四、逻辑操作符

逻辑操作符用来组合多个条件。

### 4.1 and（与）

**同时满足多个条件**

```css
/* 宽度在 768px 到 1023px 之间 */
@media (min-width: 768px) and (max-width: 1023px) {
  /* 只有平板 */
}

/* 横屏且宽度大于 768px */
@media (min-width: 768px) and (orientation: landscape) {
  /* 平板/电脑横屏 */
}
```

### 4.2 or（或）

**满足任意一个条件**（用逗号表示）

```css
/* 宽度小于 768px 或 高度小于 500px */
@media (max-width: 767px), (max-height: 499px) {
  /* 小屏幕或低高度 */
}

/* 等价于 */
@media (max-width: 767px), (max-height: 499px) {
  /* 手机或横屏手机 */
}
```

### 4.3 not（非）

**不满足条件**

```css
/* 不是打印设备 */
@media not print {
  .nav {
    display: block;
  }
}

/* 宽度不小于 768px */
@media not (max-width: 767px) {
  /* 等价于 min-width: 768px */
}
```

### 4.4 only（仅）

**防止旧浏览器不识别媒体查询时应用样式**

```css
/* 只有支持媒体查询的浏览器才应用 */
@media only screen and (min-width: 768px) {
  /* 现代 */
}
```

> 💡 `only` 对现代浏览器没有影响，主要是为了兼容老版本 IE

## 五、常用断点

### 5.1 常见断点标准

```css
/* ========== 手机 ========== */
@media (max-width: 767px) {
  /* 手机竖屏 */
}

/* ========== 平板 ========== */
@media (min-width: 768px) and (max-width: 1023px) {
  /* 平板 */
}

/* ========== 电脑 ========== */
@media (min-width: 1024px) {
  /* 电脑 */
}
```

### 5.2 Bootstrap 断点

| 断点 | 屏幕宽度 | 设备类型 |
|-----|---------|---------|
| xs | < 576px | 手机 |
| sm | ≥ 576px | 大手机 |
| md | ≥ 768px | 平板 |
| lg | ≥ 992px | 小电脑 |
| xl | ≥ 1200px | 电脑 |
| xxl | ≥ 1400px | 大电脑 |

```css
/* Bootstrap 风格断点 */
@media (min-width: 576px) { /* sm */ }
@media (min-width: 768px) { /* md */ }
@media (min-width: 992px) { /* lg */ }
@media (min-width: 1200px) { /* xl */ }
@media (min-width: 1400px) { /* xxl */ }
```

### 5.3 Tailwind CSS 断点

```css
/* Tailwind 风格（移动优先）*/
@media (min-width: 640px) { /* sm */ }
@media (min-width: 768px) { /* md */ }
@media (min-width: 1024px) { /* lg */ }
@media (min-width: 1280px) { /* xl */ }
@media (min-width: 1536px) { /* 2xl */ }
```

## 六、移动优先 vs 桌面优先

### 6.1 移动优先（推荐）

**先写手机样式，再用 min-width 扩展到大屏**

```css
/* ========== 默认：手机样式 ========== */
.container {
  padding: 10px;
  font-size: 14px;
}

.nav {
  flex-direction: column;
}

/* ========== 平板 ========== */
@media (min-width: 768px) {
  .container {
    padding: 20px;
    font-size: 16px;
  }

  .nav {
    flex-direction: row;
  }
}

/* ========== 电脑 ========== */
@media (min-width: 1024px) {
  .container {
    max-width: 1200px;
    margin: 0 auto;
  }
}
```

**优点：**
- ✅ 手机用户加载更少 CSS
- ✅ 代码更简洁
- ✅ 符合渐进增强原则

### 6.2 桌面优先

**先写电脑样式，再用 max-width 适配小屏**

```css
/* ========== 默认：电脑样式 ========== */
.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 20px;
}

.nav {
  flex-direction: row;
}

/* ========== 平板 ========== */
@media (max-width: 1023px) {
  .container {
    max-width: 100%;
    padding: 15px;
  }
}

/* ========== 手机 ========== */
@media (max-width: 767px) {
  .container {
    padding: 10px;
  }

  .nav {
    flex-direction: column;
  }
}
```

**缺点：**
- ❌ 手机用户需要加载更多 CSS
- ❌ 后面写的样式会覆盖前面的，代码可读性较差

## 七、实际应用示例

### 示例1：响应式布局

```css
/* 手机：单列 */
.grid {
  display: grid;
  grid-template-columns: 1fr;
  gap: 10px;
}

/* 平板：两列 */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
    gap: 20px;
  }
}

/* 电脑：三列 */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
    gap: 30px;
  }
}
```

### 示例2：导航栏响应

```css
.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 10px;
}

/* 手机：汉堡菜单 */
@media (max-width: 767px) {
  .nav-menu {
    display: none;  /* 隐藏菜单 */
    position: absolute;
    top: 100%;
    left: 0;
    right: 0;
    flex-direction: column;
    background: white;
  }

  .nav-menu.active {
    display: flex;  /* 点击后显示 */
  }

  .hamburger {
    display: block;  /* 显示汉堡图标 */
  }
}

/* 电脑：正常菜单 */
@media (min-width: 768px) {
  .nav-menu {
    display: flex;
    flex-direction: row;
  }

  .hamburger {
    display: none;  /* 隐藏汉堡图标 */
  }
}
```

### 示例3：响应式字体

```css
html {
  font-size: 16px;  /* 基准字体 */
}

h1 {
  font-size: 1.5rem;  /* 24px */
}

/* 平板 */
@media (min-width: 768px) {
  html {
    font-size: 18px;
  }

  h1 {
    font-size: 2rem;  /* 36px */
  }
}

/* 电脑 */
@media (min-width: 1024px) {
  html {
    font-size: 20px;
  }

  h1 {
    font-size: 2.5rem;  /* 50px */
  }
}
```

### 示例4：响应式图片

```css
img {
  max-width: 100%;  /* 永远不超出容器 */
  height: auto;     /* 保持比例 */
}

/* 懒加载占位图 */
img {
  background: #f0f0f0;
  min-height: 200px;
}
```

### 示例5：深色模式

```css
:root {
  --bg-color: #ffffff;
  --text-color: #000000;
  --accent-color: #3498db;
}

/* 深色模式 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-color: #1a1a1a;
    --text-color: #ffffff;
    --accent-color: #5dade2;
  }
}

body {
  background: var(--bg-color);
  color: var(--text-color);
}

.button {
  background: var(--accent-color);
  color: var(--bg-color);
}
```

### 示例6：打印样式

```css
/* 默认屏幕样式 */
.nav,
.sidebar,
.footer {
  display: block;
}

/* 打印时 */
@media print {
  /* 隐藏不需要的元素 */
  .nav,
  .sidebar,
  .footer,
  .button {
    display: none !important;
  }

  /* 确保内容可见 */
  .main-content {
    display: block !important;
    width: 100% !important;
  }

  /* 移除背景色节省墨水 */
  * {
    background: white !important;
    color: black !important;
  }

  /* 设置页边距 */
  @page {
    margin: 2cm;
  }
}
```

## 八、最佳实践

### 8.1 使用相对单位

```css
/* ❌ 不推荐：固定像素 */
.container {
  width: 320px;
}

/* ✅ 推荐：百分比 */
.container {
  width: 100%;
  max-width: 1200px;
}

/* ✅ 推荐：rem/em */
h1 {
  font-size: 2rem;  /* 相对于根元素字体 */
}
```

### 8.2 图片和视频响应式

```css
/* 图片永远不超出容器 */
img,
video {
  max-width: 100%;
  height: auto;
  display: block;
}
```

### 8.3 视口元标签

**别忘了在 HTML 中添加：**

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

**含义：**
- `width=device-width` — 宽度等于设备宽度
- `initial-scale=1.0` — 初始缩放比例为 1

### 8.4 测试响应式

```css
/* 调试时显示当前断点 */
body::before {
  content: "手机 (< 768px)";
  position: fixed;
  top: 10px;
  left: 10px;
  background: rgba(0, 0, 0, 0.7);
  color: white;
  padding: 5px 10px;
  border-radius: 4px;
  z-index: 9999;
}

@media (min-width: 768px) {
  body::before {
    content: "平板 (768px - 1023px)";
  }
}

@media (min-width: 1024px) {
  body::before {
    content: "电脑 (≥ 1024px)";
  }
}
```

### 8.5 避免常见错误

```css
/* ❌ 错误：缺少空格 */
@media(min-width:768px) { }

/* ✅ 正确：@media 后有空格 */
@media (min-width: 768px) { }

/* ❌ 错误：用 px 表示断点 */
@media (min-width: 768px) and (max-width: 1024px) { }

/* ✅ 正确：断点不重叠 */
@media (min-width: 768px) and (max-width: 1023px) { }
```

## 九、CSS 变量 + 媒体查询

用 CSS 变量让代码更简洁：

```css
:root {
  --font-size-base: 16px;
  --font-size-h1: 2rem;
  --spacing-unit: 10px;
  --container-max: 100%;
}

@media (min-width: 768px) {
  :root {
    --font-size-base: 18px;
    --font-size-h1: 2.5rem;
    --spacing-unit: 15px;
    --container-max: 720px;
  }
}

@media (min-width: 1024px) {
  :root {
    --font-size-base: 20px;
    --font-size-h1: 3rem;
    --spacing-unit: 20px;
    --container-max: 960px;
  }
}

/* 使用变量 */
body {
  font-size: var(--font-size-base);
}

h1 {
  font-size: var(--font-size-h1);
}

.container {
  max-width: var(--container-max);
  margin: 0 auto;
  padding: var(--spacing-unit);
}
```

| 方案 | 手机 | 平板 | 电脑 |
|-----|-----|-----|-----|

| Bootstrap | < 576px | 576-991px | ≥ 992px |
