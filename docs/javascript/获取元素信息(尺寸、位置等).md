### 一。样式信息
* `element.style`: 内联样式，只能读取行内`style`设置的样式
* `getComputedStyle(element)`: 计算之后的样式(即最终的样式)，返回一个对象，包含所有的css属性
```javascript
const box = document.getElementById('box');
// 行内样式
console.log(box.style.backgroundColor);
// 计算后样式
const styles = getComputedStyle(box);
console.log(styles.fontSize);

```
### 二。尺寸
|属性|包含内容|计算公式（标准盒模型下）|
|:--:|:--:|:--:|
|`offsetWidth/offsetHeight`|内容区 + 内边距 + 边框 + 滚动条 |width + paddin * 2 + border * 2 + 滚动条占的宽度|
|`clientWidth/clientHeight`|内容区 + 内边距 （不包含边框和滚动条）| width + padding|
|`scrollWidth/scrollHeight`|内容实际大小 + 内边距 |若内容溢出，则大于 clientWidth/clientHeight|
```js
const div = document.getElementById('box');
console.log(div.offsetWidth);  // 元素占用的总宽度（含边框）
console.log(div.clientWidth);  // 内部可视宽度（不含边框）
console.log(div.scrollWidth);  // 滚动内容的完整宽度
```
### 三。元素位置属性
* `offsetParent`: 当前元素最近的具有定位属性的父元素，如果没有，则为 `<body>` 或 `<html>`
#### 1. `offsetTop`和`offsetLeft`
* 定义： 当前元素相对于其 `offsetParent` 的偏移量
* 取值： 当前元素的**外边框**到`offsetParent`的**内边框**
```js
const child = document.getElementById('child');
console.log(child.offsetTop);    // 相对于 offsetParent 上边的距离
console.log(child.offsetLeft);   // 相对于 offsetParent 左边的距离
```
#### 2. `clientTop`和`clientLeft`
* 定义：元素边框`（border）`的宽度
```js
const box = document.getElementById('box');
console.log(box.clientTop);   // 上边框宽度（若无边框则为0）
```
### 四。`getBoundingClientRect()`
* 定义：返回一个 `DOMRect` 对象，提供元素相对于视口`（viewport）` 的位置和尺寸
* 属性：`top`, `left`, `right`, `bottom`, `width`, `height`, x, y
* 特点：
  * 返回小数（亚像素精度），适合精确布局
  * 包含边框和内边距，相当于 offsetWidth/offsetHeight 对应的尺寸
  * 坐标相对于当前视口（滚动后会变化）。要获得相对于整个文档的坐标，需加上 `window.scrollX/scrollY`
  * 每次调用都会触发重排（重新计算布局），频繁调用影响性能
```js
const rect = element.getBoundingClientRect();
console.log(rect.top, rect.left);     // 相对于视口左上角
console.log(rect.width, rect.height); // 同 offsetWidth/offsetHeight

// 获取相对于文档的坐标
const docTop = rect.top + window.scrollY;
const docLeft = rect.left + window.scrollX;
  ```
### 五。滚动相关属性
|属性|含义|可读写|
|:--:|:--:|:--:|
|`element.scrollTop`|元素内容向上滚动的距离(被卷去的高度)|✅ 可写|
|`element.scrollLeft`|元素内容向左滚动的距离(被卷去的像素数)|✅ 可写|
|`window.scrollX (或 pageXOffset)`|文档水平滚动的距离|只读|
|`window.scrollY (或 pageYOffset)`|文档垂直滚动的距离|只读|
|`window.scrollTo()`,`scrollBy()`,`scrollIntoView()`|控制滚动的方法|
```js
// 获取滚动距离
const scrollTop = document.documentElement.scrollTop || document.body.scrollTop;
const scrollLeft = window.scrollX;

// 设置滚动位置
element.scrollTop = 100;            // 让元素内部滚动到指定位置
window.scrollTo(0, 500);            // 滚动文档到 (0, 500)
element.scrollIntoView({ behavior: 'smooth', block: 'center' });
```
### 六。综合示例与常见场景
#### 1. 场景1：判断元素是否在视口内（可见性检测）
```js
function isInViewport(element) {
  const rect = element.getBoundingClientRect();
  return (
    rect.top >= 0 &&
    rect.left >= 0 &&
    rect.bottom <= window.innerHeight &&
    rect.right <= window.innerWidth
  );
}
```
#### 2. 获取元素相对于文档的绝对位置
```js
const rect = element.getBoundingClientRect();
const offset = {
  top: rect.top + window.scrollY,
  left: rect.left + window.scrollX
};
```
#### 3. 场景3：获取元素内部可视区域尺寸（不含滚动条和边框）
```js
const innerWidth = element.clientWidth;
const innerHeight = element.clientHeight;
```
#### 4.获取滚动条宽度（检测系统滚动条占位）
```js
const scrollbarWidth = window.innerWidth - document.documentElement.clientWidth;
```
### 七。重要注意事项总结
* 读取 `offset` 族属性或 `getBoundingClientRect()` ,`element.innerText`会强制浏览器同步重排
* `offsetWidth`永远等于元素在**页面上占用**的总宽度
* 元素或其祖先 `display: none`，所有尺寸位置属性均为 0，`getBoundingClientRect()` 也返回全零矩形
