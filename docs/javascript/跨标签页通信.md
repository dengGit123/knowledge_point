# 浏览器跨标签页通信

## 简介

跨标签页通信是指在同一浏览器中，不同标签页之间进行数据传递和消息交互的能力。

## 适用场景

- 用户在多个标签页同时登录/登出，需要同步状态
- 一个标签页的数据变化需要通知其他标签页
- 多标签页共享用户操作状态
- 多窗口应用的状态同步

## 实现方式对比

| 方式 | 同源限制 | 跨域支持 | 兼容性 | 数据量 | 推荐度 |
|------|----------|----------|--------|--------|--------|
| BroadcastChannel | ✅ | ❌ | 现代浏览器 | 大 | ⭐⭐⭐⭐⭐ |
| localStorage | ✅ | ❌ | 全部 | 小(~5MB) | ⭐⭐⭐⭐ |
| sessionStorage | ✅ | ❌ | 全部 | 小 | ⭐⭐⭐ |
| SharedWorker | ✅ | ❌ | 现代浏览器 | 大 | ⭐⭐⭐⭐ |
| Service Worker | ✅ | ❌ | 现代浏览器 | 大 | ⭐⭐⭐⭐ |
| postMessage + window.open | ❌ | ✅ | 全部 | 大 | ⭐⭐⭐⭐⭐ |

## 1. Broadcast Channel API

### 简介

专门为跨标签页通信设计的 API，简单易用，现代浏览器推荐方式。

### 兼容性

| Chrome | Firefox | Safari | Edge | Node.js |
|--------|---------|--------|------|---------|
| 54+ | 38+ | 15.4+ | 79+ | ❌ |

### 基本语法

```javascript
// 创建频道
const channel = new BroadcastChannel('channel-name');

// 发送消息
channel.postMessage(data);

// 接收消息
channel.onmessage = (event) => {
  console.log(event.data);
};

// 关闭频道
channel.close();
```

### 使用场景示例：用户登录状态同步

```javascript
// 用户在任一标签页登出时，通知其他标签页
class AuthSync {
  constructor() {
    this.channel = new BroadcastChannel('auth-sync');
    this.init();
  }

  init() {
    // 监听登出事件
    this.channel.onmessage = (event) => {
      if (event.data.type === 'logout') {
        this.handleLogout();
      }
    };
  }

  // 登出时通知其他标签页
  logout() {
    this.channel.postMessage({ type: 'logout' });
    this.handleLogout();
  }

  handleLogout() {
    // 清除本地存储
    localStorage.removeItem('token');
    localStorage.removeItem('user');

    // 跳转到登录页
    window.location.href = '/login';
  }

  close() {
    this.channel.close();
  }
}

// 使用
const authSync = new AuthSync();
// authSync.logout();
```

## 2. Storage Event（localStorage）

### 简介

利用 `storage` 事件，当 localStorage 发生变化时，其他同源页面会收到通知。

### 兼容性

| Chrome | Firefox | Safari | Edge |
|--------|---------|--------|------|
| 全部 | 全部 | 全部 | 全部 |

### 基本语法

```javascript
// 页面 A - 设置数据
localStorage.setItem('key', JSON.stringify({ data: 'value' }));

// 页面 B - 监听变化
window.addEventListener('storage', (event) => {
  console.log('旧值:', event.oldValue);
  console.log('新值:', event.newValue);
  console.log('键名:', event.key);
  console.log('URL:', event.url);
});
```

### 注意事项

1. **storage 事件只在其他页面触发**，当前页面不会收到自己的事件
2. 只能监听同一域名下的 localStorage 变化
3. 数据被序列化为字符串，需要手动 JSON.parse
4. 适合简单数据传递，不适合频繁通信

## 3. SharedWorker

### 简介

SharedWorker 是一种可以被多个页面共享的 Worker，所有页面都可以与同一个 Worker 通信。

### 兼容性

| Chrome | Firefox | Safari | Edge |
|--------|---------|--------|------|
| ✅ | ✅ | ❌ | ✅ |

### 实现步骤

#### 步骤 1：创建 Worker 脚本

```javascript
// shared-worker.js
const connections = [];

// 监听连接
self.onconnect = (event) => {
  const port = event.ports[0];
  connections.push(port);

  // 监听消息
  port.onmessage = (e) => {
    const { type, data } = e.data;

    // 广播给所有连接的页面
    connections.forEach((conn) => {
      conn.postMessage({
        type,
        data,
        timestamp: Date.now()
      });
    });
  };

  // 连接建立时发送欢迎消息
  port.postMessage({
    type: 'connected',
    data: { count: connections.length },
    timestamp: Date.now()
  });

  // 连接关闭时清理
  port.onclose = () => {
    const index = connections.indexOf(port);
    if (index > -1) {
      connections.splice(index, 1);
    }
  };

  // 启动端口
  port.start();
};
```

```html
    worker.port.onmessage = (event) => {

      const time = new Date(timestamp).toLocaleTimeString();

          userCount.textContent = data.count;

    // 添加消息到列表

      messages.scrollTop = messages.scrollHeight;

    document.getElementById('message').addEventListener('keypress', (e) => {
      if (e.key === 'Enter') sendMessage();
    });

    // 页面关闭时通知
    window.addEventListener('beforeunload', () => {
      worker.port.postMessage({ type: 'disconnect' });
    });

    addMessage('正在连接 Worker...', 'system');
  </script>
</body>
</html>
```

## 4. postMessage + window.open

### 简介

用于通过 `window.open()` 打开的窗口与父窗口之间的通信，支持跨域。

### 兼容性

```html
        return;
      }

      childWindow = window.open(
        './child.html',

        'width=500,height=500,left=100,top=100'
      );

      childStatus.textContent = '已打开';
      addMessage('✅ 子页面已打开');
    }

    // 发送消息给子页面
    function sendToChild() {
      if (!childWindow || childWindow.closed) {
        addMessage('❌ 子页面未打开');
        return;
      }

      const message = {
        type: 'greeting',
        data: {
          text: '你好，来自父页面的问候！',
          timestamp: Date.now()
        }
      };

      childWindow.postMessage(message, '*');
      addMessage(`📤 发送: ${message.data.text}`);
    }

    // 发送颜色指令
    function sendColor() {
      if (!childWindow || childWindow.closed) {
        addMessage('❌ 子页面未打开');
        return;
      }

      const colors = ['#fecaca', '#bbf7d0', '#bfdbfe', '#fde68a'];
      const randomColor = colors[Math.floor(Math.random() * colors.length)];

      childWindow.postMessage({
        type: 'change-color',
        data: { color: randomColor }
      }, '*');

      addMessage(`📤 发送颜色: ${randomColor}`);
    }

    // 关闭子页面
    function closeChild() {
      if (childWindow && !childWindow.closed) {
        childWindow.close();
        addMessage('🔒 子页面已关闭');
        childStatus.textContent = '已关闭';
      }
    }

    // 监听来自子页面的消息
    window.addEventListener('message', (event) => {
      // 安全检查：验证消息来源
      // if (event.origin !== 'http://localhost:3000') return;

      const { type, data } = event.data;

      switch (type) {
        case 'loaded':
          addMessage('📥 子页面已加载完成');
          break;
        case 'response':
          addMessage(`📥 收到回复: ${data.text}`);
          break;
        case 'ping':
          addMessage('📥 收到 ping，回复 pong...');
          event.source.postMessage({
            type: 'pong',
            data: { time: Date.now() }
          }, event.origin);
          break;
        default:
          addMessage(`📥 未知消息类型: ${type}`);
      }
    });

    // 检查子页面状态
    setInterval(() => {
      if (childWindow && childWindow.closed) {
        childStatus.textContent = '已关闭';
        childWindow = null;
      }
    }, 1000);

    function addMessage(msg) {
      const time = new Date().toLocaleTimeString();
      messages.innerHTML += `[${time}] ${msg}<br>`;
      messages.scrollTop = messages.scrollHeight;
    }
  </script>
```

## 5. Service Worker + Message Channel

### 简介

使用 Service Worker 作为消息中转站，实现跨标签页通信。

### 完整示例

#### Service Worker 脚本

```javascript
// sw.js
const CACHE_NAME = 'v1';

self.addEventListener('install', (event) => {
  console.log('Service Worker 安装完成');
  self.skipWaiting();
});

self.addEventListener('activate', (event) => {
  console.log('Service Worker 激活');
  event.waitUntil(self.clients.claim());
});

// 监听来自页面的消息
self.addEventListener('message', (event) => {
  const { type, data } = event.data;

  // 广播给所有客户端
  self.clients.matchAll().then((clients) => {
    clients.forEach((client) => {
      // 不发送给自己
      if (client.id !== event.source.id) {
        client.postMessage({
          type,
          data,
          from: event.source.id
        });
      }
    });
  });
});
```

## 方式选择建议

```javascript
// 根据场景选择通信方式
function chooseCommMethod(requirements) {
  const {
    needCrossDomain,    // 是否需要跨域
    needLargeData,      // 是否传输大数据
    needRealtime,       // 是否需要实时通信
    browserSupport      // 浏览器兼容性要求
  } = requirements;

  if (needCrossDomain) {
    return 'postMessage + window.open';
  }

  if (browserSupport === 'legacy') {
    return 'localStorage';
  }

  if (needRealtime && needLargeData) {
    return 'SharedWorker';
  }

  // 默认推荐
  return 'BroadcastChannel';
}
```

## 完整的通信封装类

```javascript
/**
 * 跨标签页通信封装类
 * 自动选择最佳通信方式
 */
class CrossTabMessenger {
  constructor(channelName, options = {}) {
    this.channelName = channelName;
    this.options = options;
    this.listeners = new Map();
    this.channel = null;

    this.init();
  }

  init() {
    // 优先使用 BroadcastChannel
    if (typeof BroadcastChannel !== 'undefined') {
      this.channel = new BroadcastChannel(this.channelName);
      this.channel.onmessage = (event) => this.handleMessage(event);
      return;
    }

    // 降级到 localStorage
    if (typeof window !== 'undefined') {
      window.addEventListener('storage', (event) => {
        if (event.key === `ctm_${this.channelName}` && event.newValue) {
          try {
            const data = JSON.parse(event.newValue);
            this.handleMessage({ data });
          } catch (e) {
            console.error('Failed to parse message:', e);
          }
        }
      });
    }
  }

  // 发送消息
  send(type, data) {
    const message = { type, data, timestamp: Date.now() };

    if (this.channel instanceof BroadcastChannel) {
      this.channel.postMessage(message);
    } else {
      localStorage.setItem(`ctm_${this.channelName}`, JSON.stringify(message));
    }
  }

  // 监听消息
  on(type, callback) {
    if (!this.listeners.has(type)) {
      this.listeners.set(type, []);
    }
    this.listeners.get(type).push(callback);
  }

  // 取消监听
  off(type, callback) {
    if (!this.listeners.has(type)) return;

    const callbacks = this.listeners.get(type);
    const index = callbacks.indexOf(callback);
    if (index > -1) {
      callbacks.splice(index, 1);
    }
  }

  // 处理收到的消息
  handleMessage(event) {
    const { type, data } = event.data;
    const callbacks = this.listeners.get(type) || [];

    callbacks.forEach((callback) => {
      try {
        callback(data, event);
      } catch (e) {
        console.error(`Error in ${type} handler:`, e);
      }
    });
  }

  // 销毁
  destroy() {
    if (this.channel) {
      this.channel.close();
    }
    this.listeners.clear();
  }
}

// 使用示例
const messenger = new CrossTabMessenger('app-events');

// 监听事件
messenger.on('user:logout', (data) => {
  console.log('用户登出:', data);
  window.location.href = '/login';
});

messenger.on('cart:update', (data) => {
  console.log('购物车更新:', data);
  updateCartUI(data);
});

// 发送事件

messenger.send('cart:update', { items: [...], count: 5 });
```
