### 原型链
* 对象的属性和方法的查找规则
* 查找规则： 先在当前对象中查找，如果没有找到，就去它的原型对象中找，如果还没有找到，就继续去原型的原型中找，直到找到`Object.prototype.__proto__`为止，如果在`Object.prototype.__proto__`依然没有找到，则返回undefined

### 原型链和作用域链
|特性| 原型链 | 作用域链 |
|:---| :---- | :------ |
|作用对象| 对象的属性和方法 | 函数作用域中的变量 |
|查找规则|从当前对象开始，沿着原型链向上查找直到`Object.prototype.__proto__`为止|从当前作用域开始，沿着作用域链向上查找直到全局作用域为止|
|顶层对象|`Object.prototype.__proto__` |全局作用域|
|确认时机|创建对象时自动确定，可以修改|函数定义时就确定了，并被存储在函数的内部属性[[scope]]中|

### 构造函数,原型对象的关系
1. 函数有一个`prototype`属性，指向原型对象
2. 原型对象有一个`constructor`属性，指向构造函数

### instanceof关键字
* 用来判断一个对象是否属于某个构造函数
* 原理：沿着原型链查找，如果在某个地方找到了构造函数`prototype`属性指向的对象，则返回true
* 对象有`__proto__`属性，指向构造函数的原型对象
```js
function Person() {}  
var person = new Person();
person instanceof Person; // true
```
