### 一. offsetParent
* **作用**: 获取距离该元素**最近的已定位**(position不为static)的祖先元素
* **特殊情况**‌：如果父级元素都没有定位，则返回`body`；如果元素`display`为`none`，则返回`null`
### 二. offsetWidth与offsetHeight
* 计算方式： ` offsetWidth` = `width`(样式设置) + 左右`padding` + 左右`border`;
* 特点: **包含**元素的‌内容宽度，内边距(`padding`),边框(`border`),**不包含**外边距(`margin`),**滚动条**；
* 返回值‌：纯数字，单位为像素，会四舍五入为整数
### 三. offsetLeft与offsetTop
* 含义: `offsetLeft`返回元素左外边框至`offsetParent`左内边框的距离
### 四. ‌clientWidth与clientHeight
* ‌计算方式‌: `clientWidth` = `width`(样式设置) + 左右`padding` - 垂直滚动条宽度
* 特点: **仅包含**元素的‌内容宽度‌和‌左右内边距（`padding`）,  不包含边框，减去滚动条宽度
### 五. ‌clientTop与clientLeft
* 通常对应元素的上边框和左边框宽度

### 六. ‌scrollWidth与scrollHeight
* 含义‌：返回元素内容的实际宽度和高度，包括被滚动条隐藏的部分(**即元素的实际尺寸，包括滚动条隐藏的部分**)
### 七. ‌scrollTop与scrollLeft
* 作用‌：获取或设置元素内容垂直/水平滚动的像素数
