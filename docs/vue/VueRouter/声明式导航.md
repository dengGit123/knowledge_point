### 一、概述

`<RouterLink>` 是 Vue Router 提供的**声明式导航组件**，用于替代原生的 `<a>` 标签实现页面跳转。它不会触发浏览器刷新，而是由 Router 在内部处理路由切换。

> 📖 [Vue Router - RouterLink](https://router.vuejs.org/zh/api/interfaces/RouterLinkProps.html)

### 二、基本用法

```vue
<template>
  <!-- 字符串路径 -->
  <RouterLink to="/home">首页</RouterLink>

  <!-- 绑定对象（推荐） -->
  <RouterLink :to="{ name: 'home' }">首页</RouterLink>

  <!-- 带查询参数 -->
  <RouterLink :to="{ path: '/search', query: { q: 'vue' } }">搜索</RouterLink>

  <!-- 带路径参数 -->
  <RouterLink :to="{ name: 'user', params: { id: 1 } }">用户</RouterLink>
</template>
```

### 三、激活状态样式

当当前路由匹配到 `<RouterLink>` 的目标路径时，会自动添加 CSS 类名：

```vue
<template>
  <!-- 默认会添加 router-link-active（包含匹配）和 router-link-exact-active（精确匹配） -->
  <RouterLink to="/home">首页</RouterLink>
</template>

<style>
/* 包含匹配：当前路径以 /home 开头时生效 */
a.router-link-active {
  color: blue;
}

/* 精确匹配：当前路径完全等于 /home 时生效（更常用） */
a.router-link-exact-active {
  color: red;
}
</style>
```

自定义激活类名：

```vue
<template>
  <RouterLink
    to="/home"
    active-class="is-active"
    exact-active-class="is-exact-active"
  >
    首页
  </RouterLink>
</template>
```

| 属性 | 默认值 | 说明 |
| :--: | :--: | :--: |
| `active-class` | `router-link-active` | 包含匹配时的 CSS 类名 |
| `exact-active-class` | `router-link-exact-active` | 精确匹配时的 CSS 类名 |

> 💡 **包含匹配 vs 精确匹配：** 如果当前路径是 `/home/about`，`<RouterLink to="/home">` 的 `active-class` 会生效（包含匹配），但 `exact-active-class` 不会（不完全等于 `/home`）。

### 四、replace 属性

默认导航会产生新的历史记录（可以后退），加上 `replace` 后会**替换当前记录**（不能后退）：

```vue
<template>
  <RouterLink to="/login" replace>登录</RouterLink>
</template>
```

### 五、custom 插槽

使用 `custom` 属性完全自定义渲染，配合 `v-slot` 获取导航状态：

```vue
<template>
  <RouterLink to="/home" custom v-slot="{ navigate, isActive, isExactActive }">
    <li
      :class="{ active: isActive, exact: isExactActive }"
      @click="navigate"
    >
      首页
    </li>
  </RouterLink>
</template>
```

| v-slot 属性 | 类型 | 说明 |
| :--: | :--: | :--: |
| `href` | `string` | 解析后的 URL |
| `isActive` | `boolean` | 是否包含匹配 |
| `isExactActive` | `boolean` | 是否精确匹配 |
| `navigate` | `function` | 触发导航的函数 |
| `route` | `RouteLocation` | 解析后的路由对象 |

### 六、常用属性一览

| 属性 | 类型 | 说明 |
| :--: | :--: | :--: |
| `to` | `string / object` | 目标路径（必填） |
| `replace` | `boolean` | 替换历史记录 |
| `active-class` | `string` | 包含匹配的类名 |
| `exact-active-class` | `string` | 精确匹配的类名 |
| `custom` | `boolean` | 自定义渲染模式 |
| `aria-current` | `string` | 无障碍属性 |
