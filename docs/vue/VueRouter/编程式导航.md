### 一、概述

除了用 `<RouterLink>` 在模板中声明式导航，Vue Router 还提供了**编程式导航 API**，通过 JavaScript 代码控制路由跳转。适用于按钮点击、条件判断后跳转、登录后重定向等场景。

> 📖 [Vue Router - 编程式导航](https://router.vuejs.org/zh/guide/essentials/navigation.html)

### 二、获取 Router 实例

```vue
<script setup>
import { useRouter } from 'vue-router'

const router = useRouter()

// 现在可以使用 router.push / replace / go 等
</script>
```

### 三、router.push —— 跳转

产生新的历史记录，用户可以后退：

```js
// 字符串路径
router.push('/home')

// 命名路由 + 参数
router.push({ name: 'user', params: { id: 1 } })

// 带查询参数
router.push({ path: '/search', query: { q: 'vue', page: 1 } })

// 带 hash
router.push({ path: '/docs', hash: '#section-1' })
```

> ⚠️ **注意：** `params` 只能配合 `name` 使用，不能配合 `path`：

```js
// ❌ params 会被忽略
router.push({ path: '/user', params: { id: 1 } })

// ✅ 使用 name
router.push({ name: 'user', params: { id: 1 } })

// ✅ 或者直接写在 path 里
router.push('/user/1')
```

### 四、router.replace —— 替换

跳转但**替换当前历史记录**，用户不能后退到当前页面：

```js
router.replace('/login')
router.replace({ name: 'home' })

// <RouterLink> 的 replace 等效写法
// <RouterLink to="/login" replace>登录</RouterLink>
```

### 五、router.go / back / forward —— 历史记录

```js
// 前进/后退指定页数（正数前进，负数后退）
router.go(1)    // 前进 1 页
router.go(-1)   // 后退 1 页
router.go(3)    // 前进 3 页（如果历史记录不够则静默失败）

// 快捷方法
router.back()     // 后退一页（等同 go(-1)）
router.forward()  // 前进一页（等同 go(1)）
```

### 六、导航返回值

`router.push` 和 `router.replace` 返回一个 **Promise**，可以判断导航是否成功：

```js
async function handleLogin() {
  await loginForm.submit()
  try {
    await router.push({ name: 'dashboard' })
    // 导航成功
  } catch (error) {
    // 导航失败（被守卫拦截、重复导航等）
    console.error('导航失败:', error)
  }
}
```

### 七、API 一览

| 方法 | 说明 | 产生历史记录 |
| :--: | :--: | :--: |
| `router.push(location)` | 跳转到指定路由 | ✅ |
| `router.replace(location)` | 跳转并替换当前记录 | ❌ |
| `router.go(delta)` | 前进/后退 N 页 | — |
| `router.back()` | 后退一页 | — |
| `router.forward()` | 前进一页 | — |

### 八、常见用法

```js
// 登录成功后跳转到之前想去的页面
const route = useRoute()
const redirect = route.query.redirect || '/'
await router.push(redirect)

// 条件跳转
function handleSubmit() {
  if (isValid.value) {
    router.push({ name: 'success' })
  }
}

// 跳转后刷新数据
async function goToUser(id) {
  await router.push({ name: 'user', params: { id } })
  await refreshData()
}
```
