### 一、概述

路由配置是 Vue Router 的基础，定义了 **URL 路径与 Vue 组件之间的映射关系**。通过一个 `routes` 数组，告诉 Router 访问什么 URL 时该渲染哪个组件。

> 📖 [Vue Router - 起步](https://router.vuejs.org/zh/guide/)

### 二、安装

```bash
npm install vue-router@4
```

### 三、基本配置

#### 1. 创建路由实例

```js
// router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import HomeView from '@/views/HomeView.vue'
import AboutView from '@/views/AboutView.vue'

const routes = [
  { path: '/', component: HomeView },
  { path: '/about', component: AboutView },
]

const router = createRouter({
  history: createWebHistory(),
  routes,
})

export default router
```

#### 2. 注册到 Vue 应用

```js
// main.js
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'

createApp(App).use(router).mount('#app')
```

#### 3. 在 App.vue 中使用

```vue
<!-- App.vue -->
<template>
  <nav>
    <RouterLink to="/">首页</RouterLink>
    <RouterLink to="/about">关于</RouterLink>
  </nav>
  <!-- 路由匹配到的组件渲染在这里 -->
  <RouterView />
</template>
```

### 四、路由记录结构

每一条路由规则称为一个**路由记录**，支持以下字段：

```js
{
  path: '/users',                // URL 路径
  name: 'users',                 // 路由名称（可选，推荐）
  component: UsersView,          // 匹配时渲染的组件
  children: [],                  // 嵌套子路由
  redirect: '',                  // 重定向
  alias: '',                     // 路径别名
  meta: { requiresAuth: true },  // 路由元信息
  beforeEnter: (to, from) => {}, // 路由独享守卫
  props: false,                  // 是否将 params 作为 props 传入
  sensitive: false,              // 是否区分大小写（默认 false）
  strict: false,                 // 是否严格匹配末尾斜杠（默认 false）
}
```

| 字段 | 类型 | 说明 |
| :--: | :--: | :--: |
| `path` | `string` | URL 路径，支持动态参数 `:param` |
| `name` | `string` | 路由名称，用于命名导航 |
| `component` | `Component` | 渲染的 Vue 组件 |
| `components` | `object` | 命名视图，多组件渲染 |
| `children` | `array` | 嵌套子路由 |
| `redirect` | `string/object/function` | 重定向目标 |
| `alias` | `string/array` | 路径别名 |
| `meta` | `object` | 自定义元信息 |
| `beforeEnter` | `function/array` | 路由独享守卫 |
| `props` | `boolean/function/object` | 参数传入组件的方式 |

### 五、目录结构建议

```
src/
├── router/
│   └── index.js          # 路由配置
├── views/                 # 路由级组件
│   ├── HomeView.vue
│   ├── AboutView.vue
│   └── UserView.vue
└── components/            # 可复用组件
    └── Header.vue
```

> 💡 **提示：** 路由级组件放在 `views/` 目录，可复用组件放在 `components/` 目录，这是常见约定。
