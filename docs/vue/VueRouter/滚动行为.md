### 一、概述

当用户在路由之间切换时，页面默认会停留在当前滚动位置。通过配置 `scrollBehavior`，可以控制路由切换后页面**滚动到什么位置**。

> 📖 [Vue Router - 滚动行为](https://router.vuejs.org/zh/guide/advanced/scroll-behavior.html)

### 二、基本配置

在创建路由实例时传入 `scrollBehavior` 函数：

```js
const router = createRouter({
  history: createWebHistory(),
  routes,
  scrollBehavior(to, from, savedPosition) {
    // 返回滚动位置
    return { top: 0 }
  },
})
```

### 三、返回值类型

| 返回值 | 效果 |
| :--: | :--: |
| `{ top: 0 }` | 滚动到页面顶部 |
| `{ top: 0, left: 0 }` | 滚动到指定坐标 |
| `{ el: '#section' }` | 滚动到指定元素 |
| `{ el: '#section', behavior: 'smooth' }` | 平滑滚动到元素 |
| `savedPosition` | 恢复浏览器历史中的滚动位置 |
| `false` | 不滚动 |

### 四、常见用法

#### 1. 默认滚回顶部

```js
scrollBehavior() {
  return { top: 0 }
}
```

#### 2. 恢复历史滚动位置

浏览器前进/后退时恢复之前的滚动位置，新导航则回到顶部：

```js
scrollBehavior(to, from, savedPosition) {
  if (savedPosition) {
    return savedPosition // 后退时恢复位置
  }
  return { top: 0 } // 新导航回到顶部
}
```

#### 3. 滚动到锚点

```js
scrollBehavior(to) {
  if (to.hash) {
    return {
      el: to.hash,
      behavior: 'smooth', // 平滑滚动
    }
  }
  return { top: 0 }
}
```

#### 4. 滚动到指定元素

```js
scrollBehavior() {
  return {
    el: '#main-content', // 滚动到 id 为 main-content 的元素
    top: 50,             // 偏移 50px
    behavior: 'smooth',
  }
}
```

#### 5. 异步滚动

返回 Promise 实现延迟滚动（等过渡动画完成后再滚动）：

```js
scrollBehavior(to, from, savedPosition) {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve({ top: 0 })
    }, 300) // 等待 300ms（过渡动画时间）
  })
}
```

### 五、完整配置示例

```js
const router = createRouter({
  history: createWebHistory(),
  routes,
  scrollBehavior(to, from, savedPosition) {
    // 1. 浏览器前进/后退 → 恢复位置
    if (savedPosition) {
      return savedPosition
    }

    // 2. 有锚点 → 滚动到锚点
    if (to.hash) {
      return { el: to.hash, behavior: 'smooth' }
    }

    // 3. 默认 → 滚回顶部
    return { top: 0 }
  },
})
```
