### 一、概述

路由元信息（`meta`）是路由记录上的**自定义数据字段**，可以为每条路由附加任意信息，如权限标记、页面标题、是否需要缓存等。在导航守卫和组件中都可以通过 `route.meta` 访问。

> 📖 [Vue Router - 路由元信息](https://router.vuejs.org/zh/guide/advanced/meta.html)

### 二、定义 meta

```js
const routes = [
  {
    path: '/',
    component: HomeView,
    meta: {
      title: '首页',
      requiresAuth: false,
    },
  },
  {
    path: '/dashboard',
    component: DashboardView,
    meta: {
      title: '控制台',
      requiresAuth: true,
      roles: ['admin', 'editor'],
    },
  },
  {
    path: '/admin',
    component: AdminView,
    meta: {
      title: '管理后台',
      requiresAuth: true,
      roles: ['admin'],
    },
  },
]
```

### 三、在导航守卫中使用

最常见的用法是在全局前置守卫中做权限校验：

```js
router.beforeEach((to) => {
  // 设置页面标题
  document.title = to.meta.title || '默认标题'

  // 权限校验
  if (to.meta.requiresAuth && !isLoggedIn()) {
    return { name: 'login', query: { redirect: to.fullPath } }
  }

  // 角色校验
  if (to.meta.roles) {
    const userRoles = getUserRoles()
    const hasRole = to.meta.roles.some((role) => userRoles.includes(role))
    if (!hasRole) {
      return { name: 'forbidden' }
    }
  }
})
```

### 四、在组件中使用

```vue
<script setup>
import { useRoute } from 'vue-router'
const route = useRoute()

console.log(route.meta.title)        // '控制台'
console.log(route.meta.requiresAuth) // true
</script>
```

### 五、TypeScript 类型扩展

在 TypeScript 项目中，可以通过模块扩展为 `meta` 添加类型提示：

```ts
// router/index.ts
import 'vue-router'

// 扩展 RouteMeta 类型
declare module 'vue-router' {
  interface RouteMeta {
    title?: string
    requiresAuth?: boolean
    roles?: string[]
  }
}

// 之后使用 route.meta 时会有完整的类型提示
```

### 六、嵌套路由的 meta 合并

嵌套路由的 `meta` 字段会从父到子**合并**（子路由覆盖父路由的同名字段）：

```js
const routes = [
  {
    path: '/dashboard',
    meta: { requiresAuth: true }, // 父路由 meta
    children: [
      {
        path: 'settings',
        meta: { title: '设置' }, // 子路由 meta
        // 实际 meta = { requiresAuth: true, title: '设置' }
      },
    ],
  },
]
```

可以通过 `route.matched` 遍历所有匹配记录来获取完整的 meta：

```js
router.beforeEach((to) => {
  // 检查所有匹配记录中是否有 requiresAuth
  if (to.matched.some((record) => record.meta.requiresAuth)) {
    if (!isLoggedIn()) {
      return '/login'
    }
  }
})
```
