
### VS Code 插件
| 插件 | 功能 |
|:--:|:--:|
| `Bookmarks`（代码书签） | 在代码行中添加书签标记，允许快速跳转 |
| `Project Manager`（项目管理器） | 帮助开发者高效管理多个项目，实现一键快速切换 |
| `Path Intellisense`（路径自动补全） | 在输入文件或文件夹路径时提供智能补全建议 |
| `Error Lens`（错误透镜） | 将代码中的错误、警告等信息直接内联显示在当前代码行的末尾 |
| `Import Cost` | 在 `import` 语句旁实时显示引入模块的树摇后体积 |
| `indent-rainbow`（彩虹缩进） | 用不同颜色交替显示代码的缩进，让代码结构层次分明，提高可读性 |
| `Better Comments`（更好的注释） | 为注释文本提供不同颜色和高亮样式 |
| `Auto Rename Tag`（自动重命名标签） | 在修改 `HTML`/`XML` 等标记语言的开始标签时，自动同步修改对应的闭合标签 |
| `Auto Close Tag`（自动闭合标签） | 当输入开始标签的闭合尖括号 `>` 时，自动补全对应的结束标签 |
| `JavaScript (ES6) Code Snippets`（代码片段） | 为 `JavaScript` 和 `TypeScript` 提供大量 `ES6` 语法的代码片段 |
| `GitLens`（Git 增强） | 增强 `VS Code` 内置的 `Git` 功能，提供代码作者追溯、提交历史搜索、行级对比等可视化能力 |
| `EditorConfig for VS Code`（编辑器配置） | 通过项目根目录下的 `.editorconfig` 文件，在团队内或跨不同编辑器间统一代码风格（如缩进、换行符等） |
| `Codeium`（AI 编码助手） | 免费的 `AI` 编程助手，提供代码自动补全、聊天式代码生成与解释、自然语言重构等能力 |
