## 🔥 高频面试题速查 Top 10

| # | 面试题 | 一句话答案 | 详见 |
|---|--------|-----------|------|
| 1 | 盒模型是什么？ | 元素由 content → padding → border → margin 四层组成，`border-box` 的 width 包含前三者 | [盒模型](./盒模型.md) |
| 2 | BFC 是什么？ | 独立渲染区域，内部布局不影响外部，可解决 margin 合并、高度塌陷 | [BFC](./BFC.md) |
| 3 | 如何实现居中？ | Flex（`justify-content + align-items`）、Grid（`place-items`）、`absolute + transform` | [居中方案](./居中方案.md) |
| 4 | CSS 选择器优先级怎么算？ | 行内(1000) > ID(100) > 类/伪类(10) > 元素(1)，逐级比较 | [选择器优先级](./选择器优先级.md) |
| 5 | `flex: 1` 和 `flex: auto` 的区别？ | `flex: 1` = `1 1 0%`（忽略内容均分），`flex: auto` = `1 1 auto`（按内容分配） | [Flex布局](./Flex布局.md) |
| 6 | 回流和重绘的区别？ | 回流 = 重新计算布局（昂贵），重绘 = 重新绘制像素（较便宜），回流必然触发重绘 | [重绘回流](./重绘回流与性能优化.md) |
| 7 | CSS 变量和 SCSS 变量的区别？ | CSS 变量运行时存在可动态修改，SCSS 变量编译时固定 | [CSS变量](./CSS变量与预处理器.md) |
| 8 | 如何实现响应式？ | 媒体查询 + 弹性布局 + vw/rem 适配，推荐 Mobile-first | [响应式设计](./响应式设计与移动端适配.md) |
| 9 | 清除浮动有哪些方式？ | clearfix 伪元素、`overflow: hidden`、`display: flow-root` | [浮动](./浮动与清除浮动.md) |
| 10 | CSS 动画如何优化性能？ | 只动画 `transform` 和 `opacity`（GPU 合成层），避免动画布局属性 | [动画](./动画与过渡.md) |
