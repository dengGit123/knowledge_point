# 居中方案

> 官方文档：[MDN - 居中对齐](https://developer.mozilla.org/zh-CN/docs/Web/CSS/CSS_flexible_box_layout/Aligning_items_in_a_flex_container)

---

## 一、核心概念

CSS 居中是面试最高频的题目之一。根据场景不同（水平/垂直/水平+垂直、行内/块级、已知/未知宽高），有多种方案。

---

## 二、水平居中

### 2.1 行内元素 / 行内块元素

```css
.parent {
  text-align: center;
}
```

```html
<div class="parent">
  <span>行内元素居中</span>
  <button>按钮居中</button>
  <img src="..." alt="图片居中">
</div>
```

### 2.2 定宽块级元素

```css
.child {
  width: 200px;
  margin: 0 auto;
}
```

### 2.3 不定宽块级元素

```css
/* 方案 1：flex */
.parent {
  display: flex;
  justify-content: center;
}

/* 方案 2：fit-content + margin */
.child {
  width: fit-content;
  margin: 0 auto;
}

/* 方案 3：绝对定位 + transform */
.parent { position: relative; }
.child {
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}
```

### 水平居中方案对比

| 方案 | 适用元素 | 需要固定宽高 | 兼容性 |
|------|---------|-------------|--------|
| `text-align: center` | 行内/行内块 | ❌ | ✅ 全兼容 |
| `margin: 0 auto` | 定宽块级 | ✅ 需 width | ✅ 全兼容 |
| `flex + justify-content` | 任意 | ❌ | ✅ 现代浏览器 |
| `absolute + transform` | 任意 | ❌ | ✅ 现代浏览器 |

## 三、垂直居中

### 3.1 行内元素

```css
/* 单行文字 */
.parent {
  height: 100px;
  line-height: 100px; /* 等于容器高度 */
}

/* 多行文字 */
.parent {
  display: table-cell;
  vertical-align: middle;
}
```

### 3.2 块级元素

```css
/* 方案 1：flex（推荐） */
.parent {
  display: flex;
  align-items: center;
}

/* 方案 2：绝对定位 + transform */
.parent { position: relative; }
.child {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}

/* 方案 3：绝对定位 + margin: auto（需固定宽高） */
.parent { position: relative; }
.child {
  position: absolute;
  top: 0;
  bottom: 0;
  margin: auto 0;
  height: 100px; /* 必须有固定高度 */
}

/* 方案 4：table-cell */
.parent {
  display: table-cell;
  vertical-align: middle;
  height: 200px;
}
```

### 垂直居中方案对比

| 方案 | 需要固定宽高 | 对文档流影响 | 兼容性 |
|------|-------------|-------------|--------|
| `line-height` | ❌ | 无 | ✅ 全兼容（仅单行文字） |
| `flex + align-items` | ❌ | 无 | ✅ 现代浏览器 |
| `absolute + transform` | ❌ | 脱离文档流 | ✅ 现代浏览器 |
| `absolute + margin: auto` | ✅ | 脱离文档流 | ✅ 全兼容 |
| `table-cell + vertical-align` | ❌ | 改变 display | ✅ 全兼容 |

## 四、水平 + 垂直居中（面试重点）

### 4.1 方案一：Flex（⭐ 最推荐）

```css
.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

✅ 优点：代码最少，不需要固定宽高，兼容性好

### 4.2 方案二：Grid（最简洁）

```css
.parent {
  display: grid;
  place-items: center; /* justify-items + align-items */
}
```

✅ 优点：一行代码搞定

### 4.3 方案三：绝对定位 + transform

```css
.parent { position: relative; }

.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

✅ 优点：不需要固定宽高
⚠️ 缺点：脱离文档流，`transform` 可能影响子元素的 `fixed` 定位

### 4.4 方案四：绝对定位 + margin: auto（需固定宽高）

```css
.parent { position: relative; }

.child {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  margin: auto;
  width: 200px;
  height: 100px;
}
```

✅ 优点：兼容性好
⚠️ 缺点：必须设置固定宽高

### 4.5 方案五：绝对定位 + 负 margin（需固定宽高）

```css
.parent { position: relative; }

.child {
  position: absolute;
  top: 50%;
  left: 50%;
  width: 200px;
  height: 100px;
  margin-top: -50px;  /* height 的一半 */
  margin-left: -100px; /* width 的一半 */
}
```

⚠️ 缺点：必须知道精确宽高

### 4.6 方案六：table-cell

```css
.parent {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
  width: 500px;
  height: 300px;
}

.child {
  display: inline-block;
}
```

✅ 优点：兼容 IE8+
⚠️ 缺点：需要额外设置宽高

### 全部方案对比

| 方案 | 代码量 | 需固定宽高 | 脱离文档流 | 兼容性 | 推荐度 |
|------|--------|-----------|-----------|--------|--------|
| **Flex** | ⭐ 少 | ❌ | ❌ | ✅ 好 | ⭐⭐⭐⭐⭐ |
| **Grid** | ⭐ 最少 | ❌ | ❌ | ✅ 好 | ⭐⭐⭐⭐ |
| **absolute + transform** | 少 | ❌ | ✅ | ✅ 好 | ⭐⭐⭐ |
| **absolute + margin auto** | 中 | ✅ | ✅ | ✅ 最好 | ⭐⭐ |
| **absolute + 负 margin** | 多 | ✅ | ✅ | ✅ 最好 | ⭐⭐ |
| **table-cell** | 多 | ❌ | ❌ | ✅ IE8+ | ⭐⭐ |

## 五、常见面试题

### Q1：请说出尽可能多的居中方案？

**答：**
- **水平居中**：`margin: 0 auto`、`text-align: center`、`flex justify-content: center`
- **垂直居中**：`line-height`、`flex align-items: center`、`absolute + transform`
- **水平+垂直**：`flex`（推荐）、`grid place-items: center`、`absolute + transform`、`absolute + margin: auto`

### Q2：不确定宽高怎么居中？

**答：** 三种方案：① `display: flex; justify-content: center; align-items: center;`（推荐）；② `display: grid; place-items: center;`；③ `position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);`。三种都不需要知道宽高。

### Q3：`margin: 0 auto` 居中的条件？

**答：** 三个条件：① 元素必须是**块级元素**（或 `display: block`）；② 必须有明确的 **`width`**（不能是 `width: auto` 或 `100%`）；③ 不能脱离文档流（`float`、`absolute`、`fixed` 下无效）。

### Q4：绝对定位 + margin: auto 居中的原理？

**答：** 当绝对定位元素同时设置 `top: 0; right: 0; bottom: 0; left: 0;` 时，浏览器会将其拉伸填满父元素。但如果元素有固定宽高，`margin: auto` 会自动计算上下左右的 margin，使其居中。这是 CSS 规范定义的行为——当绝对定位元素在某个方向上的尺寸确定时，`auto` margin 用于分配剩余空间。

## 六、注意事项

1. **Flex 居中兼容性**：IE11 有少量 Flex bug，现代浏览器完全支持
2. **transform 居中的副作用**：`transform` 会创建新的层叠上下文，影响内部 `fixed` 定位
3. **`margin: 0 auto` 不适用于 `float` 元素**：浮动元素不支持 `margin: auto` 水平居中
4. **Grid 居中是最佳现代方案**：`place-items: center` 一行代码，但 Flex 兼容性更广
