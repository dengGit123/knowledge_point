### 索引查询（Indexed Access Types）
* 索引查询允许我们通过索引来查找类型的特定属性或元素类型
```typescript
type PropertyType = Type[Key];
```
### 二、对象属性查询
#### 1. 单属性查询 - `Type['key']`
```typescript
interface User {
  id: number;
  name: string;
  age: number;
  address: {
    city: string;
    country: string;
  };
}

// 查询单个属性类型
type UserId = User['id'];        // number
type UserName = User['name'];    // string
type UserAge = User['age'];      // number

// 实际使用
const userId: UserId = 123;
const userName: UserName = "Alice";
```
#### 2. 嵌套属性查询 - `Type['key1']['key2']`
```typescript
// 查询嵌套属性类型
type CityType = User['address']['city'];      // string
type CountryType = User['address']['country']; // string

// 链式查询
type DeepProperty = User['address']['city'];  // string
```
### 三、联合类型查询
#### 1. 多个键查询 - 联合类型
```typescript
// 查询多个属性的联合类型
type UserBasicInfo = User['id' | 'name' | 'age'];  // number | string

// 相当于
type UserBasicInfo2 = User['id'] | User['name'] | User['age'];
```
#### 2. 使用 `keyof` 查询所有属性
```typescript
// 查询所有属性的联合类型
type AllUserTypes = User[keyof User];
// 等价于：number | string | { city: string; country: string; }

// 排除某些类型
type NonStringProperties = User[Exclude<keyof User, 'name'>];
// 等价于：number | { city: string; country: string; }
```

### 四、注意事项
1. 索引查询的类型 `K` 必须是 `keyof T` 的子类型，否则会报错
2. 当` T `是联合类型时，索引查询会返回每个成员类型中对应属性的类型的联合
