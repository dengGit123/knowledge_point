### 一. 基础工具类型
#### 1 `Partial<T>` - 将所有属性变为可选
```typescript
interface User {
  id: number;
  name: string;
  age: number;
}

type PartialUser = Partial<User>;
// 等价于 { id?: number; name?: string; age?: number; }

// 实现原理
type MyPartial<T> = {
  [P in keyof T]?: T[P];
};
```
#### 2 `Required<T>` - 将所有属性变为必选
```typscript
interface User {
  id?: number;
  name?: string;
}

type RequiredUser = Required<User>;
// 等价于 { id: number; name: string; }

// 实现原理
type MyRequired<T> = {
  [P in keyof T]-?: T[P];
};
```
#### 3 `Readonly<T>` - 将所有属性变为只读
```typescript
type ReadonlyUser = Readonly<User>;
// 等价于 { readonly id: number; readonly name: string; }

// 实现原理
type MyReadonly<T> = {
  readonly [P in keyof T]: T[P];
};
```
#### 4 `Record<K, T>` - 构造一个对象类型，其属性键为K，属性值为T
```typescript
type UserRecord = Record<string, User>;
// 等价于 { [key: string]: User }

type UserById = Record<number, User>;
// 等价于 { [key: number]: User }

// 实现原理
type MyRecord<K extends keyof any, T> = {
  [P in K]: T;
};
```
### 二。选择与排除工具类型
#### 1. `Pick<T, K>` - 从T中选择一组属性K
```typescript
type UserBasic = Pick<User, "id" | "name">;
// 等价于 { id: number; name: string; }

// 实现原理
type MyPick<T, K extends keyof T> = {
  [P in K]: T[P];
};
```
#### 2. `Omit<T, K>` - 从T中排除一组属性K
```typescript
type UserWithoutAge = Omit<User, "age">;
// 等价于 { id: number; name: string; }

// 实现原理（使用 Exclude）
type MyOmit<T, K extends keyof any> = Pick<T, Exclude<keyof T, K>>;
```
#### 3. `Exclude<T, U>` - 从 T 中排除可以赋值给 U 的类型 (T,U是一个类型)
```typescript
type T0 = Exclude<"a" | "b" | "c", "a">;      // "b" | "c"
type T1 = Exclude<string | number | boolean, number>; // string | boolean
type T2 = Exclude<User | null | undefined, null | undefined>; // User
```
#### 4. `Extract<T, U>` - 从 T 中提取可以赋值给 U 的类型 (T,U是一个类型)
```typescript
type T0 = Extract<"a" | "b" | "c", "a" | "f">; // "a"
type T1 = Extract<string | number | boolean, number>; // number
```
### 3. 函数相关工具类型
#### 1. `Parameters<T>` - 获取函数类型的参数类型
```typescript
type Func = (name: string, age: number) => void;

type Params = Parameters<Func>;  // [string, number]
type FirstParam = Params[0];     // string

// 实现原理
type MyParameters<T extends (...args: any) => any> = 
  T extends (...args: infer P) => any ? P : never;
```
#### 2. `ReturnType<T>` - 获取函数类型的返回值类型
```typescript
type Func = () => User;
type Return = ReturnType<Func>;  // User

// 实现原理
type MyReturnType<T extends (...args: any) => any> = 
  T extends (...args: any) => infer R ? R : any;
```
#### 3. `ConstructorParameters<T>` - 获取构造函数类型的参数类型
```typescript
class UserClass {
  constructor(public id: number, public name: string) {}
}

type CtorParams = ConstructorParameters<typeof UserClass>; // [number, string]
```
#### 4. `InstanceType<T>` - 获取构造函数类型的实例类型
```typescript
type UserInstance = InstanceType<typeof UserClass>; // UserClass
```
### 4. 条件与推断工具类型
#### 1. `NonNullable<T>` - 排除null和undefined
```typescript
type T0 = NonNullable<string | number | null | undefined>; // string | number
```
### 5. 字符串操作工具类型
#### 1. `Uppercase<T>` - 将字符串中的所有字母转换为大写
```typescript
type T0 = Uppercase<"hello">; // "HELLO"
```
#### 2. `Lowercase<T>` - 将字符串中的所有字母转换为小写
```typescript
type T0 = Lowercase<"Hello">; // "hello"
```
#### 3. `Capitalize<T>` - 将字符串的第一个字母转换为大写
```typescript
type T0 = Capitalize<"hello">; // "Hello"
```
#### 4. `Uncapitalize<T>` - 将字符串的第一个字母转换为小写
```typescript
type T0 = Uncapitalize<"Hello">; // "hello"
```
### 6. 自定义字符串模板类型
```typescript
// 将属性名转换为 getter 方法名
type Getters<T> = {
  [K in keyof T as `get${Capitalize<string & K>}`]: () => T[K];
};

interface User {
  name: string;
  age: number;
}

type UserGetters = Getters<User>;
// { getName: () => string; getAge: () => number; }
```
