# 生成文档技术栈限制

**重要规则：在生成任何技术文档时，必须遵守以下技术栈限制。**

## 允许的技术栈
- javascript
- TypeScript
- css
- scss
- html
- vue
- vite
- webpack

## 禁止的技术栈
- **React** - 严禁生成 React 相关示例代码
- **Angular** - 严禁生成 Angular 相关示例代码
- **Svelte** - 严禁生成 Svelte 相关示例代码
- 其他非 Vue 的前端框架

## 说明
本项目专注于 Vue 技术栈，所有代码示例、配置说明、实现方案都应基于 Vue 3、TypeScript、Vite、Webpack 等技术。
